cash_payment_page

 Top 

cash_payment_page - displays instructions about cash payment gateway

 

Parameters

•payment_id - ID of a payment
•item_name - description of an action. Payment will be taken for this action.
•amount - amount of payment  

 

Templates

•cash_gateway.tpl - the template for the Cash gateway with the description of all payment properties

Local variables

o$payment_id - the ID of a payment
o$item_name -  payment description
o$amount  - payment amount
o$user.username  -  the username of a current user

 

•wire_transfer.tpl - displays  payment properties for wire transfer

Local variables

o$payment_id -  the ID of a payment
o$item_name -  payment description
o$amount  - payment amount
o$user.username  -  the username of a current user
