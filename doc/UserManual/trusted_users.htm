Trusted Users

 Top  Previous  Next

Trusted users can activate their listings without administrator's approval and listing moderation.

 

By default, all listings of "Trusted Users" are activated immediately upon the creation, and are updated immediately if free, or upon the successful payment if paid.

 

However, even Trusted Users listings may be rejected ("Rejected" status) by admin. In this case, the user will have to update the listing. After the update, the rejected listing receives the "Pending" status even if it belongs to a Trusted User. Only the administrator can change the listing status from "Pending" to "Approved".

 

To designate a user a Trusted User, go to Admin Panel -> User Management -> Manage Users > Edit User Info. On that page, check the "Trusted User" box.

                 

 

To set/unset multiple users' status as Trusted Users, on the Admin Panel -> User Management -> Manage Users page, check/uncheck relevant users and click Make Trusted / Make Untrusted.
