credit_transactions - display credit transactions table for user (used on "My Transaction" page).

 

Parameters (parameters to this function are transaction filter data)

•sid — id of the transaction to display.
•date_from — starting date for the date range filter, default value is 30 day before the current date
•date_to — ending date for the date range filter, default value is the current date
