When you start working on your site, its content, look and add listings there, please configure the following general settings in Admin Panel:

 

1.Listing Picture Width

Listing Picture Height

Listing Thumbnail Width

Listing Thumbnail Height

You need to specify the size for images and thumbnails because iLister system resizes images when they are uploaded to a listing. In case you created listings and uploaded images there and THEN change the size for pictures and thumbnails, these new sizes will not be applied to existing images. You will have to re-upload all existing images in order to get pictures and thumbnails of a new sizing.

 

In case you changed the size of Listing Picture Width and Listing Picture Height, you need to change the size of a Slideshow player so it can accommodate listing images. You can adjust the player size in /vendor/jquery/css/basic.css:

#simplemodal-container {background-color:#fff; border:3px solid #ccc; margin: 10px 0 10px 0; height: 440px; width:440px }

 

2.Listing Picture Storage Method

It is better to keep all images in one place: either to store them in the File System, or save them in the Database. When you switch the method of picture storage, old images do not migrate to a new location.

 

3.Watermark image & Watermark opacity

You need to upload a desired image for watermark and configure its opacity before uploading pictures to listings. The watermark image is applied to a picture when it is uploaded to a listing. Thus when you change the watermark image, a new watermark will not be applied to existing listing pictures and you will need to re-upload them in order to the changes took an effect.

 

4.Specify the number of days between listing or subscription expiration and the date of related notifications sent.

 

5.Create a custom theme for your site

This will help you to avoid problems with future iLister upgrades and customizations. Please read all articles in the "Basic Concepts" section to understand how iLister works and how to use it well:

6.Edit the content of Static Pages "About Us" and "Contact Us". To change their texts go to the Admin Panel, then click "Content Management" then "Static Content" and then click on the "Edit" button across the relevant page item in the table (either "About" or "Contact").

 

7.Configure the Payment Gateways in Admin Panel -> Payment Management -> Payment Gateways.

 

8.Specify the content of the "Wire Transfer Payment" page. You can do this through Admin Panel selecting the section "Module Templates", then "Payment" and clicking the "Edit" button for the "wire_transfer.tpl" template.

 

9.Define the SEO Data Included in Listing URL and Listing Caption for each category of your site.

 

10.Get the Google Maps API key to enable map display on the Listing Details page.

 

11.Set up Facebook and Twitter connection setting in order your visitors can use their Facebook and Twitter accounts to sign in to your website.

 

12. Enable repost of listings and publication from your website on Facebook and Twitter.
