The Yandex Money Payment Gateway is an add-on and is not a part of iLister default package for distribution. It can be installed separately for your iLister-based website. Please request the installation conditions from our Sales Dept.

 

Overview

Yandex.Money is a convenient and safe payment system that allows users to make purchases securely online.

Since it was developed primarily for the operating on the territory of the Russian Federation, Yandex Money does not provide the option for currency conversion, and all sums are considered to be in Russian Rubles (RUR).

 

Obtaining Yandex Money API

In order to use the Yandex Money Gateway on your iLister-based website, you need to have a valid Yandex Money account and add there the application based on the Yandex Money API.

 

You can do this at the Yandex Money website -> Логин (Login) -> Управление счетом (Manage Account) -> click on the "Внешние приложения" header (External Applications) -> Подробнее (Read More) -> API для управления счетом (Account Management API) -> Использовать API для управления счетом (Use API to manage account) -> follow the link "Получите уникальный идентификатор своего сервиса" and complete the form:

•Название вашего приложения — API name
•Адрес вашего сайта — the URL of your website
•Redirect URI — http://your-website-url.com/system/payment_gateway_yandex_money/proceed_payment
•Использовать проверку подлинности приложения — we recommend to check this option.

 

After the form is completed, please click the button "Подтвердить" (Confirm).

 

You will see the data with:

•Идентификатор приложения — application identifier
•OAuth2 client_secret — secret word

 

 

Yandex Money Payment Gateway Configuration

The Yandex Money Payment Gateway can be configured in the Admin Panel -> Payment Management -> Payment Gateways -> Yandex Money -> Configure

 

 

 

 

Please fill in the configuration form with the following data:

•Account Number — the number of your Yandex Money account
•Client ID — paste the application identifier ("Идентификатор приложения")
•Secret Word — paste the "OAuth2 client_secret"
