search_form — displays listing search forms

 

Parameters

•listing_type_id — the optional parameter, listing type ID
•restore — the optional parameter, restore content of form fields
•form_template — the template for the search form

 

Templates

•search_form.tpl - displays a full search form (/search/)

Local variables

o$GLOBALS.site_url -  the global variable containing a full URL of the site
o$listing_type_id  -  the ID of the listing type
o$form_fields.[field_id].caption - the caption of the listing field. For example, to display the caption of the Year field it is $form_fields.Year.caption

 

•quick_search.tpl - quick search form from the main page
