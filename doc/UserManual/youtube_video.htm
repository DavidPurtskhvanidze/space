YouTube Video

 Top  Previous  Next

Description

This option enables a listing owner to place a video clip pertaining to the listing on the YouTube video share service and display this video on the listing details page. The availability and price of this feature is controlled by the site administrator via the Admin Panel.

 

Administration Manual

The YouTube video feature is configured in the listing package management interface of the Admin Panel.  The administrator can set certain price for this feature or specify it as "0" (zero) which allows users to enable it for their listings for free.

 

The YouTube video feature expires with the listing expiration.

 

The administrator has the ability to enable the YouTube video feature for any listing via the Edit Listing interface in the Admin Panel. If the administrator checks the " YouTube video" checkbox for a particular listing,  this option is activated for this listing regardless of whether this option has been paid for or not.

 

 

User Experience

In order to use the YouTube video option, the listing owner has to enter the video id into the relevant input box on the Edit Listing page, and then enable the YouTube video option for this listing.

 

The video clip's ID is entered into the YouTube Video ID field.

 

 

The YouTube video option is available for any active listing. The "Activate YouTube Video" link is displayed in the listing description on the "My Listings" page.

 

 

After clicking the link, the listing owner will be offered to make a payment equal to the cost of this option set by the administrator for this listing package. If the price for this option is set to 0 or left empty, this feature can be activated for free without any further prompts to make a payment.

 

When the "YouTube video" option is activated, the listing details page will contain a video player box with the relevant video from YouTube displayed.
