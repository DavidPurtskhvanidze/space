Image Slideshow

 Top  Previous  Next

Description

A slideshow enables a site visitor to view all of the listing images without selecting each image. Site visitors can adjust various options such as image rotation speed, can pause and resume slideshow, and switch to next or previous image.

 

Administration Manual

Users can order slideshow as a listing enhancement for an additional fee by purchasing a package with this option. The administrator enables free slideshow for listings by setting its price to zero (0) in the listing package settings. When the user creates a listing based on the package that includes a paid slideshow, the user can enable the slideshow by paying the amount indicated in the "Price for Slide Show Feature" field.

 

The slideshow option is disabled automatically when a listing expires.

 

The administrator can enable the slideshow for any listing via the Edit Listing interface of the Admin Panel. When the administrator checks the "Slide Show" checkbox, this option is activated for this listing regardless of whether it's been paid for by the listing package settings.

 

 

User Experience

Listing owners can enable the slideshow at the "My Listings" page. Under each listing that has this option, there will be the link "Activate Slide Show". When a listing owner clicks this link, he/she is offered to pay the price set in the listing package settings. The slideshow will be activated after the payment is completed. If the price for the option was set to 0, the slideshow is activated immediately when the listing is created.

 

 

After the slideshow option is activated for the listing, visitor can see the button "Slideshow" on the Listing Details page under the picture carousel. When the site visitor clicks this link, the listing details page is faded out and on top of it a slideshow player is displayed.

 

 

Slideshow player browses through the listing images showing listing pictures from the first to the last image. After it shows the last image, the player loops back to the first image.

 

Viewers can start or stop the player and flip images manually.
