The Quick Search form allows visitors to find listings with desired criteria with one click from your homepage by specifying desired keywords.

 

 

The template for Quick Search form is located in Admin Panel -> Mobile Site Layout -> Module Templates -> Classifieds Engine -> quick_search.tpl.

There you can modify the components of the Quick Search form.

 

Criterion / Field

You can add as many fields on the search form as you wish. All you need to do is to follow the rules when adding a new field. You can use the code of existing fields as an example of how to add more fields/criteria.

 

For example, the code for zip radius search is the following:

<tr>

 <td>[[FormFieldCaptions!Search Within]]:</td>                //displays the caption for the field

 <td>{search property=ZipCode}</td>                                //displays the ID of the field

</tr>

 

 

You can find the templates for all field types in the Field Types section.

 

New Search Form

In order to create a new Search Form you need to do the following main steps:

1.create new template with Search Form components in Admin Panel -> Mobile Site Layout -> Module Templates -> Classifieds Engine -> category_templates/search/;
2.create new Mobile Site Page which will display a new Search Form in Admin Panel -> Mobile Site Layout -> Site Pages -> Add a New Site Page;
3.create new template with Search Results components in Admin Panel -> Module Templates -> Classifieds Engine -> category_templates/display/;
4.create new Mobile Site Page which will display a new Search Results form in Admin Panel -> Mobile Site Layout -> Site Pages -> Add a New Site Page.

 

Please refer to the main article about new iLister search form.
