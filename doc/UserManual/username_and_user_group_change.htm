Username and User Group Change

 Top  Previous  Next

A user can change his / her username at the "My Profile" page.

 

 

 

When user specified an existing username, the website displays an error message prompting to choose another username.

 

 

The User Group can be changed both by site administrator from the "Manage Users" interface in Admin Panel and by the user in his / her profile.

 

 

When the user clicked the link to change his current User Group, he is asked to select a new group among available User Groups.
