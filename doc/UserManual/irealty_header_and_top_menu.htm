Header and Top Menu

 Top 

Header is the upper part of the page. Top and middle menus are parts of the header.

 

 

The header template file can be found in /modules/main/apps/FrontEnd/templates/{theme_name}/page_parts/header_block.tpl

 

The look of the header is specified in the PageHeader div in the CSS file which you can find at /modules/main/apps/FrontEnd/files/{theme_name}/design.css

Header.tpl of different themes uses their own background image to display header picture:

.header

{

 background-image: url("header.png");

 background-position: 0 39%;

 background-repeat: repeat-x;

 height: 166px;

}

 

Other components (logo, slogan etc.) are applied on top of the header background.

 

You can view header.tpl in Admin Panel -> Site Layout -> Page Templates -> page_parts/header_block.tpl.

There you can see the CSS styles which are used for header section and the name of header image (header_image.png):

.headerImage

{

 bottom: 0;

 position: absolute;

 right: 100px;

}

 

How to Change Header

The best way to change the header image is to give your custom header the name of existing header file, e.g. header.png for the irealty_tabs_blue theme, and place your custom header file to  the folder /modules/main/apps/FrontEnd/files/irealty_tabs_blue/

 

In order to upload your own header image, you need to connect to your server via FTP or via Control Panel (please ask your hosting administrator if your Control Panel has the option to work with file transferring).

 

The blue gradient background is specified as follows:

.header

{

 background-image: url("header.png");

}

 

 

Edit Header Size

The hight of header is specified in the CSS style which is fully shown above:

.header 

{

 height: 166px;

}

 

The width is inherited from the SiteFrame style:

#SiteFrame

{

 width: 1000px;

}

 

We do not recommend to change the width, unless your site uses a completely new custom theme which is not inherited from iLister product themes.

 

You can modify the height of the header picture in the related CSS style.

 

Please check which style is used for the header of your theme in Admin Panel -> Site Layout -> Page Templates -> page_parts/header_block.tpl.

 

Where to Modify the Top and Middle Menus

You can find the menu templates in Admin Panel -> Site Layout -> Module Templates -> Menu -> top_menu.tpl and middle_menu.tpl. There you can add menu items.

 

CSS style for the top menu is:

.topMenu

{

 color: #999999;

 font-family: verdana;

 font-weight: bold;

}

 

The style for the middle menu is:

.middleMenu

{

 background: url("middle_menu_background.png") repeat-x scroll 0 0 transparent;

 bottom: 10px;

 height: 36px;

 left: 0;

 line-height: 36px;

 position: absolute;

 width: 100%;

}

 

How to Manage Menu Items

You can remove or add a menu item in menu templates:

•top menu — Admin Panel -> Site Layout -> Module Templates -> Menu -> top_menu.tpl
•middle menu — Admin Panel -> Site Layout -> Module Templates -> Menu -> middle_menu.tpl

 

In order to add a new item, please use the below code as an example:

<li><a href="{$GLOBALS.site_url}/newpage/">[[New Page]]</a></li>

 

When you add a new item to the menu, please make sure that you have the page where this item leads, like the /newpage/ Site Page in the above example.
