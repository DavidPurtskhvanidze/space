Modify Secondary Pages Template

 Top  Previous  Next

Secondary pages are all pages excepting the homepage.

 

In accordance with default iLister Site Pages settings, almost all secondary pages use default.tpl as a template, which surrounds the content of a secondary page.

 

For example, the "Browse By State" page is a typical secondary page.

 

In Admin Panel -> Site Pages -> /browse-by-state/ you can check that it uses default.tpl as a template:

Please note that if there was no file specified in the "Template" parameter, then default.tpl is used.

 

Below you can see how the page is built on the front-end of the site. Default.tpl includes:

•headerMenuBlock with top menu, user greeting and language selector
•mainContent div including:
obrowse_items_and_results.tpl - the template file which displays the browsing results.
▪module is 'classifieds'
▪function is 'browse'
▪browse_template is 'browse_items_and_results.tpl'.
oin its turn, browse_items_and_results.tpl inclides featured_listings.tpl
•footer_block.tpl with copyright

 

 

You can learn more on how iLister builds pages in the "Page Building" section.

The "Template Identification" feature will help you to find out which template is used to display the parts of any iLister page.
