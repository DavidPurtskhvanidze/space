SEO Data Included in Listing URL

 Top  Previous  Next

Description

iLister provides administrator the ability to specify data for search engine optimization for each category. This setting is called "SEO Data Included in Listing URL" and can be found on the configuration page of a category.

 

This setting is inherited by child categories. This means that site administrator can define one SEO Data Included in Listing URL for a parent category which will be inherited by all sub-categories; and re-declare the URL SEO Data for child categories which demand different data for a better SEO results.

 

How to use SEO Data Included in Listing URL

Site administrator can use it for the link which leads to the Listing Details page for a better SEO results. For example, you can see such a link on the search results and browsing results pages.

 

In order to do so, the administrator should use the variable {$listing.urlData} in the template which displays the page with these links (please refer to the section "Template Identification").

{capture assign="listingUrl"}{$GLOBALS.site_url}/listing/{$listing.id}/{$listing.urlData|escape:"urlpathinfo"}.html?searchId={$listing_search.id}{/capture}

 

The {$listing.urlData} variable will form the link to the Listing Details page in accordance to the configuration of "SEO Data Included in Listing URL" setting in the properties of listing category.
