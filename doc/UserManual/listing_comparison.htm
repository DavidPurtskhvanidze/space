Listing Comparison

 Top  Previous  Next

This function is used to collate and compare listings based on all parameters stored in the database for the listings.

 

To compare listings, add listings to the Comparison chart by checking the "Compare Ad" checkbox on the search results page or on the listing details page.

 

The Comparison chart can be viewed at any moment. To view the Comparison chart, click the "View Comparison Results" link on the search results page -> "Manage Search" drop-down controls:

 

In the opened window, you will see all listings that are being compared.

 

The "N/A" mark means that listings of this category do not have this field and thus the data can not be applied for comparison.

 

On this page you can view the listing details page, remove the listing from chart, or remove all listings from the chart.

 

 

 

Note: listings stay in the Comparison chart for the active session only. You will need to repeat the above actions if your session has expired.
