By default, iLister does not have a tree-type field. However, you can easily create a tree-type field, for example, StateProvince. This field can be either common field, or category specific for one category. Let's make it common in our example.

 

How to Add a New Tree Level

 

Please go to Admin Panel -> Categories -> Edit Common Fields -> Add a New Common Field -> edit StateProvince.

 

When you created the field clicking on the button "Add", you need to edit the field in order to complete the parameters for tree levels.

 

When you saved the settings for levels, please edit the field again and follow the link "Edit Tree Values" in order to manage tree levels data.

 

You can add data to tree-type field in two ways:

1.you can manually fill in the form "Add a new Value"

In addition, you can sort the values in all tree levels alphabetically ascending, or sort the values within the current tree level in ABC order ascending or descending.

 

2.you can import the file with all data following the link "Import data from file". File can be either in CSV or XLS format.
