user_membership_plans

 Top 

user_membership_plans - displays membership plans that are available for user group of the user identified by user_sid parameter

 

Parameters

user_sid - user id (numeric)

 

Templates

user_membership_plans.tpl

 

Local variables

•$membership_plans - the array of all available membership plans
•$membership_plan.name - the value of the "Name" field of the membership plan
•$membership_plan.description - the value of the "Description" field of the membership plan
•$membership_plan.price - the value of the "Price" field of the membership plan
•$membership_plan.subscription_period - the value of the "Expiration Period" field of the membership plan
•$membership_plan.type - the type of the membership plan (Fee based / Subscription)
•$membership_plan.classifieds_listing_amount - the number of listings available for this membership plan.

 

Functions

•{include} -  includes another file

 

Parameters

•file - the name of included file. Includes the packagesBlock.tpl file to display all available packages for this membership plan.
•packages - the array of all available packages for this membership plan.
