New Search Form

 Top  Previous  Next

By default, iLister does not use any specific search form for a category. However, you may want to use a specific search form for one of your custom categories. Below please find the instructions on how you can do this.

 

In order to create/modify the standard Search Form you need to do the following:

1. Go to Admin Panel > Site Layout > Manage Forms

2. Add new form

3. Specify the ID of your search(for example, WantAdsSearch)

4. Select the Category(for ex. Want-Ads)

5. Add the fields that you want to be displayed on the Search form for this Category

   In order to divide your fields by subcategories(for ex. Basic Information, Features), you can add the FieldSet.

6. Create new Site Page that will display the new form (for ex. /search/Want-Ads)

 

 

In order to create a new self-designed Search Form you need to do the following main steps:

1.create new template with Search Form components;
2.specify a new template in the settings of a category, in case you want this search form to be default search form for this category;
3.create new Site Page which will display a new Search Form;
4.create new template with Search Results components;
5.specify a new template in the Search Results template setting of a category, in case you want this search results template to be default search results look for this category;
6.create new Site Page which will display a new Search Results form.

 

 

How to Create New Template for a Search Form

You can take one of the existing search forms in Admin Panel -> Site Layout -> Module Templates -> Classifieds Engine -> category_templates/search/default_search.tpl

as a basis to create new search form.

 

Here is a part of code which you need to modify:

<h1>[[Advanced search]]</h1>        // The caption for the search form

...

<form method="get" action="{$GLOBALS.site_url}/search-results/">        // The path to the page with search results for this form

    <div><input type="hidden" name="action" value="search" /></div>

    <table class="form">

        <tr>

         <td>[[Search]]</td>                                                                                 // The set of fields, specific for this search form

         <td>{search property=$form_fields.keywords.id} [[in]]                                  // The set of fields, specific for this search form

         {search property=$form_fields.category_sid.id template="category_tree.tpl"}</td>        // The set of fields, specific for this search form

        </tr>

        <tr>

            <td>[[With pictures only]]</td>

            <td>{search property=$form_fields.pictures.id}</td>

        </tr>

        <tr>

            <td>[[Search Within]]</td>

            <td>{search property=$form_fields.ZipCode.id}</td>

        </tr>

 

Let's say, we want to make a search form for a new category "RoomsForRent" that will be a child category for "Housing". The modified code of the template which we will name searchFormRoom.tpl will look as follows:

<h1>[[Find a Room]]</h1>        // The caption for the search form

...

<form method="get" action="{$GLOBALS.site_url}/search-results-rooms/">        // The path to the page with search results for this form

 <div><input type="hidden" name="action" value="search" /></div>

 <input type="hidden" name="category_sid[tree][]" value="{$category_sid}" />

 <table class="form">

         <tr>

                 <td class="searchFormCaption">[[$form_fields.State.caption]]:</td> 

                 <td>{search property=StateProvince template=tree_level_1.tpl}</td> 

         </tr>

         <tr>

                 <td class="searchFormCaption">[[$form_fields.Beds.caption]]:</td>        // The set of fields, specific for this search form

                 <td>{search property=Beds template=list.tpl}</td>                         // The set of fields, specific for this search form

         </tr>

 

When you finished the template, please connect to your server via FTP or via Control Panel (please ask your hosting administrator if your Control Panel has the option to work with file transferring) and upload the template to your server to the folder /modules/classifieds/apps/FrontEnd/templates/{theme_name}/category_templates/search.

 

How to Specify New Search Form for a Category

Go to Admin Panel -> Listing Management -> Categories -> click 'Housing' -> edit 'RoomsForRent' -> Edit 'RoomsForRent' Category Settings

 

How to Create a Site Page for New Search Form

In order to create a Site Page for this search form, you need to do the following:

1.go to Admin Panel -> Site Layout -> Site Pages -> Add a New Site Page
2.fill in all necessary fields in the Site Page form;
3.for Standard Form: choose form_manager in the "Module" field

  for self-designed Form: choose classifieds in the "Module" field

4.for Standard Form: select display_search_form in the "Function" parameter;

  for self-designed Form: select search_form in the "Function" parameter;

5.specify additional parameters like category_id and form_template;

   (for Standard Form you need to specify form_id(for ex. WantAdsSearch))

6.click the button "Save".

 

How to Create New Template with Search Results

You can take one of the existing search results forms in Admin Panel -> Site Layout -> Module Templates -> Classifieds Engine -> category_templates/display/default_search_result_item.tpl as a basis to create new search results form.

 

Here is a part of code which you need to modify:

...

{if !$listing.Description.isEmpty}

 <div class="fieldValue fieldValueDescription">{$listing.Description|strip_tags|truncate:200|nl2br}</div> // Set of fields which must be displayed on the search results page

{/if}

 

 

We need to make a search results form for the new category "RoomsForRent". The modified code of the template which we will name searchResultRooms.tpl will look as follows:

...

{if !$listing.Description.isEmpty}

 <div class="fieldValue fieldValueDescription">{$listing.Description|strip_tags|truncate:200|nl2br}</div> // Set of fields which must be displayed on the search results page

{/if}

{if !$listing.Beds.isEmpty}

 <div class="fieldValue fieldValueBeds">{$listing.Beds}</div> // Set of fields which must be displayed on the search results page

{/if}

 

When you finished the template, please upload the template to your server to the folder

/modules/classifieds/apps/FrontEnd/templates/{theme_name}/category_templates/display

 

 

How to Specify New Search Results for a Category

Go to Admin Panel -> Listing Management -> Categories -> click 'Housing' -> edit 'RoomsForRent' -> Edit 'RoomsForRent' Category Settings

 

 

How to Create Site Page for a New Search Results Form

In order to create a Site Page for this search results form, you need to do the following:

1.go to Admin Panel -> Site Layout -> Site Pages -> Add a New Site Page
2.fill in all necessary fields in the Site Page form;
3.the URI parameter must match the URI specified in the searchFormRoomsForRent.tpl template:

<form action="{$GLOBALS.site_url}/search-results-rooms/">

4.choose classifieds in the "Module" field
5.select search_results in the "Function" parameter;
6.specify additional parameters like:
•default_sorting_field
•default_sorting_order
•default_listings_per_page
•results_template
•advanced_search_form_uri — it must match the URI of the Site Page for the search form searchFormRoomsForRent.tpl
7.click the button "Add".

 

 

 

How to Create and Use FieldSet

In order to create a FieldSet, you need to do the following:

1. Go to Admin Panel > Site Layout > Manage Forms > Edit Form

   If you don't have any forms yet, you need to create standard Search Form

2. In "Edit Details" choose "FieldSet" and specify the Title. (For ex. Basic Information)

 

The FieldSet will automatically divide your search form into subcategories. You need just to put it before the fields.
