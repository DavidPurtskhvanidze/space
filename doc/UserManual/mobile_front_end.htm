Mobile Front-End

 Top  Previous  Next

The Mobile Front-End is not a part of iLister default package for distribution. It can be purchased separately for your iLister-based website for additional cost.

 

 

Overview

Mobile Front-End represents your site on mobile devices. Mobile templates are specifically designed to fit mobile screen resolution and to represent the information the best for mobile devices.

 

Mobile Front-End has a limited set of features; not all iLister features and functions are available on the mobile site.

 

The main idea of mobile site is to provide visitors with the ability to quickly find and view necessary information (listing details or seller contacts). Thus there is no registration,  listing posting, listing management and some other functions. However, all listing and seller information is visible, users can log in, post comments and save ads.

 

How It Works For Visitors

Mobile visitors of your main website are redirected to the mobile website automatically. There they can return to your main website clicking the link "Main Website" at the footer.

 

The home page has a convenient navigation on the top menu, the Quick Search form and links to browse listings — all necessary components to help visitor find what he or she needs.

 

 

Mobile URL Configuration

The URL of your mobile site can be either http://m.yoursite.com or http://yoursite.com/m (Mobile Front-End has a separate license).

The URL is configured in iLister configuration file for MFE apps/mobileFrontEnd/config/local.ini and you also should make the folder /application/m to answer to a desired URL (http://m.yoursite.com or http://yoursite.com/m) in the Control Panel of your server (please consult your server administrator on how to setup this).

 

Mobile Site Layout

Mobile Front-End has its own theme, page templates, modules and site pages.

 

Site administrator can find these templates and settings in Admin Panel -> Mobile Site Layout.

In order to learn more about mobile site modification, please refer to the section "iLister Mobile Front-End Customization" of this User Manual.
