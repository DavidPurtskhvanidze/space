Image Carousel

Description

One of the most impressive and popular web design trends is a sliding horizontal panel with switching pictures also known as Image Carousel.

 

Designers usually put this attractive element to the main page of the website, adding there the most valuable information. It can be a paid client promotion, or other important information.

 

Anyway, the Image Carousel can be and should be used as a new income item on your iLister-based website.

 

 

When site visitor put the mouse on the image, the carousel stops so the visitor can read the information and the image caption. In addition, visitors can manually switch images clicking the navigation buttons or arrows; in this case the carousel ceases the automatic image paging.

 

Image Carousel Settings

The Image Carousel settings are listed in Admin Panel -> Content Management -> Image Carousel

 

 

•Width (in px) — the width of the pictures uploaded for the Image Carousel by the website administrator (in pixels). Larger images will be resized to the specified dimensions, while smaller pictures will stay intact. When you changed the size, you need to re-upload the image so new dimensions can be applied to it.
•Height (in px) — the height of the pictures uploaded to the Image Carousel (in pixels).
•Transition time (in sec.) — the time an image is displayed (in seconds).
•Show arrows — indicates if the navigation arrows should be displayed.
•Show image numbers — specifies if the image numbers should be shown in the image navigation buttons.

 

 

Image Settings

Carousel images have their own settings:

•Caption — a tooltip displayed on the mouse over the image.
•URL — a full URL like http://www.site.com or a relative URI of your website like /contact-us/.

 

 

As you  can see, you can disable images, change the order of appearance, edit the image or its caption.

 

 

The Image Carousel Module

The look of the carousel can be easily changed in the design.css file at /application/modules/image_carousel/apps/FrontEnd/templates/_files/design.css

The arrows can be modified in /application/modules/image_carousel/apps/FrontEnd/templates/arrows.png

 

You can find the module at Admin Panel -> System Configuration -> Module Manager among other modules. There you can disable the Image Carousel module if you do not want to use it.

 

In order to insert the carousel to a page (template), please use the following code:

{module name="image_carousel" function="display_carousel"}
