Basic Theme Colors settings

 Top 

You can set colors of base blocks via Admin Panel -> Site Layout -> Themes -> Edit Colors

For Mobile Front-End you can change colors in Admin Panel -> Site Layout -> Themes -> Edit Colors

 

Main Colors:

 - Background Color

 - Text Color

 

 

 

Menu Colors:

 - Divider color

 - Background Color

 - Color of text in the menu

 

 

 

Left Menu Colors:

 - Background Color

 - Text Color

 

 

Footer Colors:

 - Background Color

 - Text Color

 

 

 

Buttons Colors:

 - Background Color

 - Text Color

 - Border color

 

 

 

Links Color:

 - Links Color

 

 

 

Also, you can add the class with specified colors to any tag, so settings will be applied to that tag as well.

For example, if you add the class "colorize-main" to <div></div> block, the background color and color of the text that were specified in Admin Panel -> Site Layout -> Themes -> Edit Colors -> Main Colors will be applied to that <div> block.
