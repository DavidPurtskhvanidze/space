The PDF Export module is an add-on and is not a part of iLister default package for distribution. It can be installed separately for your iLister-based website. Please request the installation conditions from our Sales Dept.

 

Overview

PDF Export module allows to export listings to PDF — which is known as a file format used to represent documents in a manner independent of application software, hardware, and operating systems.

 

Listings in PDF format can be easily printed, stored and shared among the visitors of your website, dealers, agents and their clients.

 

 

How It Works For Visitors

Visitors can have a hard copy of a listing in one click using the link "Export to PDF" on the Listing Details page.

 

 

After clicking the link you'll be prompted either to open it in a browser or related program; or save a .pdf file to your computer.

 

A converted listing looks great in PDF format. Please note the QR code presenting on the lower right corner.

 

 

 

How It Works For Administrator

Administrator can export one or several listings to PDF.

 

First, he can do this on the Display Listing page in the Admin Panel.

 

 

 

Second — and the most important and convenient way — he can do this on the Manage Listing page via the "Actions with selected" interface. Thus he can select one or several listings for export to PDF format.

 

 

When several ads are selected for export, the administrator will get one PDF file with the clickable table of contents and all listings.

 

 

PDF Settings

Administrator can easily change the look of main elements in the generated PDF file in the Admin Panel -> System Configuration -> System Settings -> PDF Export

 

 

 

It is very convenient to change PDF settings and check how they look in the file by clicking the link "View Example".
