Facebook Like Box

 Top  Previous  Next

iLister has integrated Facebook Social Plugin — Like Box, that enables Facebook Page owners to attract and gain likes from their own website.

 

The like box enables people using your website to do the following:

•See how many people like the Page, and which of their friends like it, too.
•Read recent posts from the Page.
•Like the Page with one click, without needing to visit the Page.

 

Please refer to the official Facebook Like Box page to get the most recent information on Like Box features and options. There you can learn how to change the width and height of the FB Like Box section, as well as how to change other parameters.

 

 

How To Manage The Like Box Section

By default, the Like Box block is visible in the iLister_tabs_blue theme only.

 

 

However, you can insert the section with FB Like Box to any other theme.

 

First, you need to open the file /modules/theme_iauto_tabs_blue/main/homepage_main_content.tpl and copy the whole FB Like Box code to a desired template:
<div id="fb-root"></div>

 <script>(function(d, s, id) {

   var js, fjs = d.getElementsByTagName(s)[0];

   if (d.getElementById(id)) return;

   js = d.createElement(s); js.id = id;

   js.src = "//connect.facebook.net/en_US/all.js#xfbml=1&appId=195402287233378";

   fjs.parentNode.insertBefore(js, fjs);

 }(document, 'script', 'facebook-jssdk'));</script>

<div class="fb-like-box" data-href="http://www.facebook.com/worksforweb.classifieds" data-width="395" data-height="250" data-show-faces="true" data-stream="false" data-show-border="false" data-header="false"></div>

 

Then you should find the code:

<div class="fb-like-box" data-href="http://www.facebook.com/worksforweb.classifieds" data-width="395" data-height="250" data-show-faces="true" data-stream="false" data-show-border="false" data-header="false">

 

and replace the URL http://www.facebook.com/worksforweb.classifieds with the URL of your Facebook page.
 

Afterwards you may need to add some styles to arrange the look of the Like Box in your theme and that's all — it will work.
