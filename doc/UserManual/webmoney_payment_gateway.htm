The WebMoney Payment Gateway is an add-on and is not a part of iLister default package for distribution. It can be installed separately for your iLister-based website. Please request the installation conditions from our Sales Dept.

 

Overview

WebMoney is a multifunctional payment tool that provides secure and immediate transactions online.

 

WebMoney does not provide the option for currency conversion, and all sums are considered to be in the currency of the merchant's purse type.

 

 

Configuring WebMoney Merchant Interface

In order to use the WebMoney (WM for short) Payment Gateway on your iLister-based website you need to have a valid WM account.

 

You can learn how to register the WebMoney account in the WebMoney Wiki page.

 

When the account is registered, you need to configure the Web Merchant Interface.

 

After login please click the link "change..." across the purse which you would like to use.

 

You will be redirected to the "Merchant purse settings" page. Please fill in the following fields:

•Secret Key — a string of characters that is added to the transfer details sent to the merchant in the notification. It is known only to the merchant and the Merchant WebMoney Transfer service.
•Result URL - http://yourwebsite.com/system/payment_gateway_webmoney/result/
•Send parameters in the preliminary inquiry (Передавать параметры в предварительном запросе) — set to "yes" (check the parameter)
•Proxy for Result URL - do not use
•Success URL - http://yourwebsite.com/system/payment/callback/webmoney/ , set as "POST"
•Fail URL and method of requesting (Fail URL и метод его вызова) - http://yourwebsite.com/system/payment/callback/webmoney/ , set as "POST"
•Let to use URL sent in form (Позволять использовать URL, передаваемые в форме) — set to "yes" (check the parameter)
•Control sign forming method (Метод формирования контрольной подписи) — select "SHA256"

 

 

Please leave other parameters intact.

 

Please note that is you selected "Test" for the "Test/Work mode" field, all your transactions will be performed in the test mode, ignoring the current option value "Test Mode" in Admin Panel -> Payment Management -> Payment Gateways -> WebMoney -> Configure

 

After the form is completed, please click the button "Save".

 

 

 

WebMoney Payment Gateway Configuration

The WebMoney Payment Gateway can be configured in the Admin Panel -> Payment Management -> Payment Gateways -> WebMoney -> Configure

 

 

 

 

Please fill in the configuration form with the following data:

•WebMoney Purse Number — the number of your WebMoney purse (wallet)
•Secret Key — the secret key which you specified in your merchant purse settings
•Test Mode — if clicked, all transactions are performed in the test mode and no actual funds will be added/withdrawn.
