Recent Ads Section

 Top 

 

In order to edit the look of the Recent Ads section you need to edit the template recent_listings.tpl in Admin Panel -> Site Layout -> Module Templates -> Recent Listings.

 

The way how recent listings are displayed can be specified by the code in page_parts\left_block.tpl:

{module name="recent_listings" function="recent_listings" recent_listings_template="recent_listings.tpl" number_of_rows="4" number_of_cols="1"}

 

You need to place this code to the template of the page where you would like to see the Recent Ads section.

 

You can change the number of rows and number of columns in the code to display as many ads as you wish.

{module name="recent_listings" function="recent_listings" recent_listings_template="recent_listings.tpl" number_of_rows="2" number_of_cols="3"}

will display

 

Please note that it's available for non-responsive themes only.

In responsive themes the number of rows and number of columns are calculated automatically depending on screen resolution.

 

Also you can specify the category for Featured Listings and Recent Listings blocks by setting "category_id":

{module name="recent_listings" function="display_recent_listings" number_of_rows="2" number_of_cols="2" category_id="Bikes"}
