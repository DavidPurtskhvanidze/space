Listing Highlight

 Top  Previous  Next

Description

This option allows a listing owner to make his/her listing stand out on the search results page. It is a great marketing tool for the listing owner, and a great money making function for the administrator. When enabled, listings are highlighted in a different color from ordinary listings. The administrator controls the availability and the cost of this feature via the Admin Panel. You can adjust how the highlight feature looks via the listing search results template.

 

 

Administration Manual

The administrator can configure the price for the Highlight feature via the listing package management interface in the Admin Panel.

 

The highlight option expires with the listing expiration.

 

The administrator has the ability to enable the highlight feature for any listing via the Edit Listing interface in Admin Panel. If the administrator checks the "Highlighted" checkbox for the listing, this option is activated for the listing regardless of whether it's been paid for or enabled in the listing package.

 

 

 

User Experience

The listing highlight option is available for any active listing. If the option was not enabled for the listing's Package by default, a "Make Highlighted" link will be displayed in the listing description on the "My Listings" page. In case the Highlight option was enabled by default, i.e. the checkbox was ticked in the settings of a Listing Package, then no link will be available to activate it.

 

 

After clicking the link, the listing owner will be offered to make a payment equal to the cost of this option set by the administrator for this listing package. If the price for this option is set to 0, this feature will be activated automatically without any further prompts to make a payment.

 

Listings that have this option enabled look different on the search result pages with their backgrounds highlighted. The exact appearance of how the listings should be highlighted (different background, frame, image, font, etc.) can be set via the Admin Panel's template management interface.
