Overview

A QR Code is a specific two-dimensional code, readable by QR barcode readers and camera phones. QR is the abbreviation for Quick Response, as the creator intended the code to allow its contents to be decoded at a high speed.

 

The code consists of black modules arranged in a square pattern on a white background. The information encoded can be text, URL or other data. The below QR code directs to the WorksForWeb website.

 

How It Works

QR code is used for mobile tagging, i.e. when mobile users can quickly learn more about the advertised item, commonly through the use of data (such as a URL) encoded in a two-dimensional barcode, meant to be read and inputted using a camera phone.

 

In order to generate a QR code a visitor should click the link "Generate QR Code" on the Listing Details page.

 

 

This will open a new page with the generated QR code image which can be saved and printed.

 

It works as follows:

1.the listing owner generates a QR code for his/her listing on your website, print the QR code and place it on different advertising website or directory/hoarding with a short description;
2.the owner of mobile device takes a picture of the QR code;
3.the mobile barcode reader decodes the taken QR code;
4.the mobile device displays the decoded URL proposing to visit your website.

 

You and any site visitor can generate a QR code of a listing and print it in order to place a code anywhere to promote the listing:

•QR codes of real estate listings can be placed to the "Home For Sale" yard sign;
•codes of automobile ads can be sticked on the advertised auto;
•the users of your site who promote their services can distribute printed QR codes everywhere.

 

 

QR Code Generator Settings

Site administrator can find the QR settings in Admin panel -> System Settings.

 

•Error correction capacity — QR Code has error correction capability. Data can be restored even if the symbol is partially dirty or damaged. A maximum 30% of codewords can be restored (Data restoration may not be fully performed depending on the amount of dirt or damage):
1.Level L        Approx. 7% of codewords can be restored.
2.Level M        Approx. 15% of codewords can be restored.
3.Level Q        Approx. 25% of codewords can be restored.
4.Level H        Approx. 30% of codewords can be restored.
•Code square size — the size of the squares units in pixels. For example, the number 4 means that the square has sides equal to 4 pixels each.
•Boundary size — the size of margins which surround a QR code in order to make it isolated on the printing or display area. The isolation makes the QR code recognition  easier for mobile devices.
