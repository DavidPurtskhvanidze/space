What Are Membership Plans

Membership plans are model contracts for user subscription for the site services. Membership plan governs which of the site services are available to the user and what are the prices for these services. User can have only one subscription to one membership plan at a time. However, users can change membership plan when current subscription expires. Administrator can create as many membership plans as desired. Please refer to the "Membership Plans" section for more information on managing site's membership plans.

 

What Membership Plans Are Made Of

Membership plans are comprised of listing packages. The listing package is a model contract for a single listing. Listing package properties define the listing activation price, whether video is allowed, the prices for other listing features like Featured Ad, Highlighting etc., how long the listing remains active and other paid or free services available for use with the listing. Listing package is assigned to listing when listing is created.

Listing packages are alternatives that user can choose between when creating a new listing. User is offered only those packages that are included to the membership plan that user has selected for subscription.

 

Relations Between Membership Plans, Listing Packages, User Groups, Users, and Listings.

•Membership plans are for users.
•Listings packages are for listings.
•User owns listings. Any user can have as many listings as allowed by membership plan user has chosen for subscription.
•Each listing holds a copy of listing package that listing owner has selected for this listing. Website user is unable to edit listing package or change listing package for listing owned. Website administrator is able to both, edit listing packages and assign new packages to existing listings.
•When user subscribes to a membership plan, only those membership plans are offered for selection that are available for the user's user group. If there is only one membership plan available for the user group, the user is subscribed to this plan without offering a choice.
•When user creates a listing, only those listing packages are offered that are available in the membership plan (contract) that user is subscribed to. If there is only one listing package available in user's membership plan, this listing package is assigned to the new listing without choice offered to the user.

Types of Membership Plans

There are two types of membership plans:

•Subscription-based membership plans — retain conditions through the whole subscription period until expiration, user's listings expire when user's subscription expires.
•Fee-based membership plans  — do not retain conditions, listings stay active until expiration date regardless whether owner's subscription is expired or not.

 

Subscription-based Membership Plans

When user subscribes to subscription-based membership plan, the conditions of this membership plan are copied into user's contract. User's contract expiration date is set according to the membership plan settings (registration date + subscription period in days). Thus, if administrator changes the settings to the subscription-based membership plan these changes will not affect the currently subscribed users. The changes will take effect only when users' subscription expires and then user selects the changed membership plan for subscription again.

However, administrator can force expiration of the user's contract using the Admin Panel.

 

Conditions retained in the user's contract include all listing packages that were available at the moment of subscription. Thus, if administrator deletes the listing package in the subscription-based membership plan, this package will still be able for use for those users that have it retained in their contract.

 

When user's contract expires, all his/her listings are deactivated.

Subscription-based membership plans can be seen as rent of a particular space on the website. Usually, subscription-based membership plans have limited number of listings (e.g. 20 listings) and include packages with free listing activation and a very long listing expiration period, which effectively means that users can have 20 active listings as long as their subscription lasts.

Subscription based-plans are useful for users that always need to have some amount of listings active, such as dealerships.

 

 

Fee-based Membership Plans

When user subscribes to a fee-based membership plan, the conditions of this membership plan are not copied into user's contract. In fact, there is no user contract for users subscribed to fee-based membership plans per se.  For such users contract is just a reference to the membership plan. Thus, if administrator changes a fee-based membership plan, the new settings are operative for all subscribed (to this plan) users immediately. However, the packages that fee-based subscribers use for their listing are retained in listings themselves until listing is deleted (deactivation does not removes the package from the listing).

 

Another important difference from subscription membership plans is that if the user's fee-based subscription expires, his/her listings will stay active until until listing reach their own expiration date. For example, if user having a fee-based subscription posts a listing 2 days before subscription expiration and chooses listing package with 30 days expiration period, the listing will stay active for 30 days, not for 2 days only.

 

Fee-based membership plans usually are free to subscribe and have a very large limit on number of listings but contain listing packages with a paid listing activation and with shorter expiration period.

Fee-based plans are good for FSBO sites, where owner does not need to have many listings active all the time. Owners usually need to have only one or two listings active until the property is sold.

 

Picking a Membership Model For Your Website

Administrator can set up just any model using the membership plan management interface. Here are some basic examples:

 

Simplest FSBO website

•A single user group — "Private Seller". Since there is only one user group, new users will not have to choose the user group at registration, which simplifies the registration.
•A single fee-based membership plan "FSBO" which is available to the only user group. Since there is only one membership plan, no membership plan selection is required.
•A single listing package "Standard" in the "FSBO" membership plan, which excludes listing package selection when listing is created making listing creation easier.
•Listing package "Standard" is $20 for activation for 30 days, $20 more to make listing featured, and $10 for any other extra (slideshow, YouTube video, highlighted).

 

Site for dealers

•A single user group — "Dealership". Since there is only one user group, new users will not have to choose the user group at registration, which simplifies the registration.
•Two subscription-based membership plans "Basic" and "Extended",  which both are available to the only user group.
