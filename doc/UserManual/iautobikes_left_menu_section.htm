Left menu is divided into two parts - left menu for visitors (guests) and a User Group specific left menu.

 

The template for the left menu for a User Group is specified in its properties in Admin Panel -> User Groups -> edit a group. By default, iLister has a specific left menu for the User Group "PrivateSeller". In case there is no user menu specified in the settings of a User Group, iLister displays common user menu which template is user_menu.tpl

 

 

A User Group menu is displayed under the common left menu if a user of this group is logged in.

 

In case there is no user menu specified in the settings of a User Group, iLister displays common user menu which template is user_menu.tpl

 

 

How to Edit a Menu Item

You can remove or add a menu item in menu templates:

•common left menu — Admin Panel -> Site Layout -> Module Templates -> Menu -> left_menu.tpl
•visitor (guest) left menu — Admin Panel -> Site Layout -> Module Templates -> Menu -> guest_menu.tpl
•common user menu — Admin Panel -> Site Layout -> Module Templates -> Menu -> user_menu.tpl
•User Group specific menus are also there, you can check the name of the template file for a User Group menu in the settings of the Group.

 

In order to add a new item, please use the below code as an example:

<li><a href="{$GLOBALS.site_url}/newpage/">[[New Page]]</a></li>

 

When you add a new item to the menu, please make sure that you have the page where this item leads, like the /newpage/ Site Page in the above example.

 

How to Add a Menu Template

You may need to have a different menu for different User Groups. To do so, you need to create a new menu template for a Group.

 

You can copy the code from one of the existing user menu templates like user_menu.tpl, and upload this file to your server at

/modules/menu/apps/FrontEnd/templates/{theme_name}/new_usergroup_menu.tpl.

Afterwards you need to specify this file new_usergroup_menu.tpl in the settings of the User Group.
