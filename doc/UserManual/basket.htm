Description

Basket is a kind of internal cart of your website. The advantages of basket are pretty obvious: the users do not need to make several burdensome payments, one for each listing created, or its option which they would like to enable.

Please note that basket works with listings and their options only (Listing Reactivation, Featured Ad, Sponsored, Highlighted, etc.), while user subscription is processed as a single payment beyond the basket via payment gateways directly.

 

 

How It Works For Users

When user created a listing, he is redirected to the Manage Listing page, where are two links, leading to the basket.

 

•View My Basket — opens a basket with all listings and their paid options.
•Pay For Listing — opens a basket filtered by the current listing.

 

If a user did not check any paid options during the listing creation, he can easily do this from the basket via the link "Add options".

 

The user will be prompted to select additional paid options.

 

If listings have some paid options enabled, the basket looks as follows:

 

Each listing has its subtotal amount and the deletion icons to remove unwanted options right from the basket.

If there is no more options to buy, there is no link "Add options".

 

When the payment is completed, the user can see the transaction on the My Transaction page. In addition, the user can see the contents of each transaction (the same information is accessible for the website administrator in the Admin Panel -> Payment Management -> Payments).
