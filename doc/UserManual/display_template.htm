display_template

 Top 

display_template - displays a content of the template specified in the template_file parameter.

 

Parameters

template_file — the template file with the content which should be displayed on the specified Site Page

 

Templates

None

 

Example of Use

You can use this function to display the block with different modules and functions.

For example, you may want to display the section with poll block and static content. You need to do the following:

1.create a template "poll_and_about.tpl" with the code specified below:

{module name="poll" function="poll_form"}

{module name="static_content" function="show_static_content" pageid="About"}

2.open the template where you would like to display this block and add the code to a desired place:

{module name="main" function="display_template" template_file="poll_and_about.tpl"}
