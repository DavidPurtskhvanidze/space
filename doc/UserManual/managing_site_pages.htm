Managing Site Pages

 Top  Previous  Next

Site Page is a set of parameters that are used for displaying any of the functions available in iLister system. Please also refer to the Site Page Building section.

 

The administrator can specify the following Site Page settings:

•URI -  the relative address under which you can find this page (e.g. /homes/). You can also set any of the file extensions here (e.g. .html, .php, .txt, .htm, etc.).
•Pass parameters via URI  - this flag means that parameters will be passed via URI . These parameters allows you to use SEO friendly URIs with the 'display_listing' and 'browse' functions. Listing ID will be passed in URI for the 'display_listing' function and browse fields will be passed in URI for the 'browse' function.
•Title – you can enter a title for your user page that will appear in the 'title' meta tag.
•Template – here you can set a template for the user page.
•Module – you can choose a module which function you want to use for this page.
•Function - each module contains one or more functions that can be used by the webmaster. Here you can specify a function to be used for this page. This function will appear instead of  the mainContent div in the chosen template.
•Parameters – additional parameters for the selected function.
•Keywords - keywords are certain words or phrases used for search engine optimization. Keywords will be input in the 'keywords' meta tag in the source of this page.
•Description -  this information will be input in the 'description' meta tag in the source of this page.

 

 

The administrator can manage Site Pages in Admin Panel -> Site Layout -> Site Pages.

 

Examples of Use

In order to add a page with static content (like "About Us" page), you need to do the following:

1.fill in all necessary fields in the Site Page form;
2.choose static_content in the "Module" field
3.select static_page in the "Function" parameter;
4.specify the page ID parameter in the "pageid" field;
5.click the button "Add".

 

In order to add a page with search form, you need to do the following:

1.fill in all necessary fields in the Site Page form;
2.choose classifieds in the "Module" field
3.select search_form in the "Function" parameter;
4.specify additional parameters like category_id and form_template;
5.click the button "Add".
