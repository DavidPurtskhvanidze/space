Watermark

A watermark is an image printed on top of every image uploaded by the users for copyright/identification/advertising purposes. The watermark is imprinted onto the uploaded image during the image upload.

 

The following image formats can be used for the watermark:

•gif
•jpg
•png

 

Please be aware though, that due to a known php bug, transparent areas of PNG-24 images may appear as black when watermark is applied to the listing picture. Whether it is reproducible or not on your website depends on the version of PHP you are using.

 

The watermark image is stored in the /files/pictures/ directory in the root of iLister's installation.

 

You can upload new watermark file from the Admin Panel -> System Configuration -> System Settings, in General settings block, the Watermark Image setting. (In order to delete current watermark, you need to check the option "Delete Watermark Picture" and click "Save" underneath the Settings table.)

 

In the same block, opacity of the watermark can be set in the Watermark opacity field. The opacity value can be set from 0 to 100.

 

 

 

 

Favicon

A favicon is an icon associated with a URL that is variously displayed, as in a browser's address bar or next to the site name in a bookmark list.

 

The following image formats can be used for the favicon:

•ICO
•PNG
•GIF

 

Please note that favicon should be 16x16 or 32x32 pixels.

 

The favicon image is stored in the /files/pictures/ directory in the root of iLister's installation.

 

You can upload new favicon file from the Admin Panel -> System Configuration -> System Settings, in General settings block, the Favicon icon setting. (In order to delete current favicon, you need to check the option "Delete Favicon Icon" and click "Save" underneath the Settings table.)
