Google AdSense

 Top  Previous  Next

As known by every website owner, Google AdSense provides a free and flexible way to earn money from websites, mobile sites, and site search results with relevant and engaging ads.

 

iLister-based websites allow to easily place your own AdSense code to get more revenue.

 

 

You can see the Google AdSense placed on the main page of iLister.

 

 

How To Use Your AdSense Code

Open the google_adsense.tpl in the Admin Panel -> Site Layout -> Page Templates and insert your AdSense code between {literal} and {/literal} tags:

{literal}

 <script type="text/javascript">        //here starts your AdSense code

 <!-- google_ad_client = "...

 </script>                                        //here it ends

{/literal}

 

 

You can insert Google AdSense to whatever page you like, using the following code on the related template:

{module name="main" function="display_template" template_file="google_adsense.tpl"}

 

For example, you can add it to the main page in the template homepage_main_content.tpl, or to the left menu in left_block.tpl.
