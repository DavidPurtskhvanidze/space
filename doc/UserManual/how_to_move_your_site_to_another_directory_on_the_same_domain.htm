How to move your site to another directory on the same domain

 Top  Previous  Next
1.Move all software files and folders to the desired directory (it can be root or a sudirectory).

 

2.Edit the apps\AdminPanel\config\local.ini and apps\FrontEnd\config\local.ini files - you need to set the correct URL there.

 

3.Apply to our Sales Dept. in order to get a new licenses for your new URL.
