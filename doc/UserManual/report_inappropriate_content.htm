Report Inappropriate Content

 Top  Previous  Next

Description

Visitors of the iLister-based website can report inappropriate content of listings and listing comments to the site administrator.

 

There is the link "Report Content" on the Listing Details page

 

Each Listing Comment and reply has the link "Report This"

 

 

How It Works

Clicking on one of the above links visitor can fill in the form with related information about the content which he or she considered to be inappropriate.

 

 

The administrator can disable CAPTCHA module for the "Report Inappropriate Content" form in Admin Panel -> System Settings -> CAPTCHA Settings.

The "Report Inappropriate Content" form will send the information to the email specified in the Notifications Email property in Admin Panel -> System Settings.

 

 

Administration Manual

Administrator receives the email with inappropriate content report containing the following useful links:

1.a link to display listing/comment in Admin Panel;
2.a link to deactivate listing or hide reported comment;
3.a link to delete listing/comment.

 

Viewing the reported listing or comment, the administrator can perform an appropriate action to react on the email:

•Edit Listing / Comment — administrator can edit the content;
•Add IP $IP to the Blocklist — the IP of the user who has left the improper content can be added to the Blocklist;
•Make the User $username Untrusted — administrator can make the author of the listing/comment content untrusted so he or she cannot post anything without administrator's approval anymore.
