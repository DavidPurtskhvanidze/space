task_scheduler

 Top 

task_scheduler - the script which runs various tasks such as system and user notifications, listing expiration, listing feed import etc. The log is written in the root directory of the system to the task_sheduler.log file.

 

The Task Scheduler data are displayed on the Admin Dashboard.

 

In addition, you can run the Task Scheduler in Admin Panel ->System Settings -> click the link 'Run Task Scheduler'

How to configure a CRON job for task_scheduler.php

 

Parameters

None

 

Templates

•task_scheduler_log.tpl - the template for the task scheduler log

 

Local variables

•$smarty.now -  the current date and time
•$expired_listings_id -  the array containing IDs of expired listings
•$listing_id -  the ID of a listing
•$expired_contracts_id -  the array containing IDs of expired contracts
•$contract_id -  the ID of a user contract
•$notified_saved_searches_id - the array containing IDs of saved searches with notifications where new listings match saved search criteria
•$search_id - the ID of a saved search
