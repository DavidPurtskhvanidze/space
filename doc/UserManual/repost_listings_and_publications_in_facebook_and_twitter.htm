Repost Listings and Publications in Facebook and Twitter

 Top  Previous  Next

Overview

The Repost feature brings only advantages and profits both for site administrator, users and advertisers: new listings are automatically reposted in administrator's account in Facebook and Twitter; site administrator also has the ability to repost publications.

 

There are Twitter repost and Facebook "Share" button available for users.

 

Listing and publication repost in social networks such as Facebook and Twitter allows to increase site popularity and grow site traffic to make it more attractive for advertisers. User listings have more demonstration and thus have more chances to find their client.

 

How It Works for Site Administrator

New active listings and publications are automatically reposted in administrator's accounts in Facebook and Twitter.

In order to turn on the Repost feature, the administrator should connect to Facebook and Twitter networks, and then enable both or either Listing Repost or Publication Repost in Admin Panel -> System Settings -> Social Networks -> Listing & News Repost.

Imported listings are not reposted (just imagine several thousands of new tweets or facebook posts!).

 

How It Works for Users

New listings of a user are automatically reposted in his/her account in Twitter account.

For Facebook users can use "Share" button that is displayed on "My listings" page.

 

In order users can enable the Twitter repost, site administrator first should configure "Listing & News Repost" in Admin Panel as shown above (in case this is not implemented, users will not see the Listing Repost section in users Profile).

 

Afterwards users can connect to Twitter networks and enable Listing Repost.

Repost settings are configured in user profile.

 

 

 

In case the repost option is enabled for listings, user can choose not to repost a listing that he's creating in Twitter network.

 

 

How It Works for Visitors

Visitors can "Like" listings so they got linked from visitors' Facebook account and can "Tweet" them so they are published in Twitter.
