The Ads On Map module is an add-on and is not a part of iLister default package for distribution. It can be installed separately for your iLister-based website. Please request the installation conditions from our Sales Dept.

 

Overview

The Ads On Map module is a great feature allowing the visitors of your iLister-based website to visualize listing search and search results. Most often it is preferable to present the information graphically instead of displaying a long list which required continuous scrolling.

 

 

Ads On Map Settings

This module works with the Google Maps API v3 only, so you need to enable it for your Google account.

 

You should specify the "Default Location" in the Admin Panel -> System Settings -> Ads On Map. You can type in a country or a city, and this place will be shown on the map when visitors start searching ads. In case of unknown location the map displays New-York.

 

 

 

How It Works For Visitors

Performing listing search with a Search Form, visitors of your website can switch to the "Find On Map" mode:

 

There they can draw a circle with a desired radius, which will be used for listing search. Visitors can change the center of the circle, move the borders of circumference, and use the quick search form, located at the map.

 

When all search criteria are set, visitors should click the button "Find" to see search results. All search results listed on a page will be displayed on a map. When visitor lists pages, the map displays new relevant results.

Visitors can switch between Map Search Results and List Search Results using two links above the map.

 

One balloon corresponds to one listing. When the mouse is on a balloon, it displays the ID of the listing and its caption.

If you click on a balloon, you'll see listing preview.

 

The Listing Details page can be opened by clicking listing preview image or its caption.
