poll_form - displays a poll block with a question and its answers

 

Parameters

None

 

Templates

Local variables

o$answer.id - the ID of an answer
o$answer.title - the text of an answer

Local variables

o$GLOBALS.site_url -  the global variable containing a full URL of the site
o$question.id - the ID of the question
o$question.title- the title of a question
o$answers - the array with the answers for the question
o$question.display - displays the question if the "Display Results" parameter is checked in the "Edit Question" block at Admin Panel -> Content Management -> Online Polls -> edit the question
