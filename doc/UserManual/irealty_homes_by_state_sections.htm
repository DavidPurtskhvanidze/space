Homes By State Section

 Top 

 

The 'Homes By State' and 'Rentals By State' sections are browsing sections.

 

Administrator can display browsing section in templates using the following code construction:

{module name="classifieds" function="browse" category_id="HomesForSale" fields="State" number_of_cols="3" browse_template="browse_by_state.tpl"}

 

In the theme irealty_tabs_blue browsing sections are shown in tabs:

<div class="browsingTabs">

 <ul>

         <li><a href="#BrowseHomesByStateTab">[[Homes by State]]</a></li>

         <li><a href="#BrowseRentalsByStateTab">[[Rentals by State]]</a></li>

 </ul>

 <div id="BrowseHomesByStateTab">

         {module name="classifieds" function="browse" category_id="HomesForSale" fields="State" number_of_cols="4"  browse_template="browse_homes_by_state.tpl"}

 </div>

 <div id="BrowseRentalsByStateTab">

         {module name="classifieds" function="browse" category_id="HomesForRent" fields="State" number_of_cols="4" browse_template="browse_rentals_by_state.tpl"}

 </div>

</div>

 

How to Edit the Browsing Sections

In order to edit the design of these section, you need to edit the files browse_homes_by_state.tpl and browse_rentals_by_state.tpl, which you can find in

Admin Panel -> Site Layout -> Module Templates -> Classifieds Engine.

You can also refer to the CSS styles which are specified in the template files for browsing.

 

The captions of the browsing blocks are translatable and can be modified in the template homepage_main_content.tpl:

<li><a href="#BrowseHomesByStateTab">[[Homes by State]]</a></li>

<li><a href="#BrowseRentalsByStateTab">[[Rentals by State]]</a></li>
