Refine Search

 Top  Previous  Next

The Refine Search is a pretty handy feature, which lets visitors to narrow their current search criteria and at the same time see the updated relevant search results.

 

 

How It Works For Visitors

The Refine Search tool is available on the search result page. Please note that it does not work with the Search by Map feature which you use by clicking the "Map" switcher on the top of search results — the refined criteria will be ignored.

 

It lists search criteria with specified values, which can be easily modified by a visitor to precise them and get more relevant search result.

 

Visitors are able to add more criteria to the Refine Search form by clicking the link "Add More Criteria" and "Show More Options" (for the boolean-type fields).

 

 

The Refine Search Form Setting And Its Modification

Each category can have its own Refine Search form. Its name is specified in the settings of a category in the Admin Panel -> Listing Management -> Categories -> edit a category -> Edit Category Settings -> Refine Search Template.

 

The website administrator can change the Refine Search form and the set of fields, specified there, in the template, indicated in the "Refine Search Template" property.

The template locates at /modules/theme_name/classifieds/category_templates/search.

 

When you add a new field to a category, it becomes visible on the Refine Search form immediately via the interface of "Add More Criteria" or "Show More Options" (for the boolean-type fields).
