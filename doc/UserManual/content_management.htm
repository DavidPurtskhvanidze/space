Content Management

 Top  Previous  Next

Besides listings and listing-related functions and pages, iLister provides the ability for the administrator to add another type of content to your site.

 

You can find the sections to manage this content in Admin Panel -> Content Management:

1.Business Catalog
2.Online Polls
3.Publications
4.Static Content

 

You can use HTML format for these pages or format them with a WYSIWYG editor. You can also add images and embedded objects in these pages.

 

 

Business Catalog

Business catalog is the list of your partners, affiliates or other companies which you want to represent on your site. Using the Business Catalog module you can join similar companies or individuals to a category (e.g. Dealerships, Realtors, Agencies, Affiliates, etc.) and create as many categories as you wish.

 

Admin Panel

 

Front-end

 

How to use it in templates

{module name="business_catalog" function="show_business_catalog" category_id="Dealerships" record_id="1"}

You can place this code in any template (like default.tpl or default_view.tpl) and it will display the information of the company "WorksForWeb" (or other company which you add, its record_id is "1", you can see it when you put your mouse over the "Edit" button next to WorksForWeb caption) of the Dealerships category. You can omit category_id and record_id parameters.

 

You can use the same parameters if you like to create a separate Site Page for Business Catalog section.

 

 

Online Polls

You can gather visitor opinions or perform other type of research by creating polls.

 

Admin Panel

 

 

Front-End

 

How to use it in templates

{module name="poll" function="poll_form"}

There are more functions of the module "poll", and they can have more parameters which you can also use.

 

You can place this code in any template (like homepage_main_content.tpl or quick_search.tpl) and you can use the same parameters if you like to create a separate Site Page for Online Poll section.

 

 

Publications

The Publications module makes a content available to visitors and users. You can publish periodic articles, news, reviews and announcements.

Also, you can add the picture to every publication in Admin Panel -> Content Management -> Publications that will be displayed in the list of publications:

 

Front-End

 

 

Admin Panel

 

 

How to use it in templates

{module name="publications" function="show_publications" category_id="News" article_id="7"}

You can place this code in a template and it will display the article "11 Nov - Our redesigned corporate website released" (its ID is "7", you can see it when you put your mouse over the "Edit" button next to article's caption) of the News category. You can omit category_id and article_id parameters.

 

You can use the same parameters if you like to create a separate Site Page for Publications section.

 

 

Static Content
