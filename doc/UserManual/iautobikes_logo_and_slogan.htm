Logo, Slogan and Keyword Search

 Top 

 

Logo

The logo of iLister-based website is located in the header of each page.

 

The file name for logotype is logo.png and it is located on your server at the iLister installation folder

/modules/main/apps/FrontEnd/templates/_files/logo.png

Please note that different themes can use different logo files and thus you need to make sure that your {theme_name} folder correlates with the theme which logo should be changed.

 

You can learn more about themes and templates in the Template System and Theme Management section.

 

You can design your own logo file and upload it to the location specified above, under the same name, i.e. logo.png.

 

In order to upload your own logo.png, you need to connect to your server via FTP or via Control Panel (please ask your hosting administrator if your Control Panel has the option to work with file transferring).

 

The position of logo file is specified by CSS style, like the following example:

.logo

{

    padding: 35px 0 0 10px;

}

 

You can find the CSS file for your theme at /modules/main/apps/FrontEnd/files/{theme_name}/design.css

 

Slogan

You can change the slogan text in Admin Panel -> Site Layout -> Page Templates -> page_parts/header_block.tpl.

<div class="slogan" >

 [[Bike Classifieds Demo Site]]

</div>

 

 

The properties of the slogan are specified as follows:

.slogan

{

    font-size: 170%;

    color: #ddd;

    top: 80px;

    left: 10px;

}

 

Keyword Search

The Keyword Search form allows visitors to find listings with desired criteria with one click from your homepage by specifying desired keywords.

 

The template for Keyword Search form is located in Admin Panel -> Site Layout -> Module Templates -> Classifieds Engine -> keyword_search.tpl.

There you can modify the components of the Keyword Search form.

 

Criterion / Field

You can add as many fields on the search form as you wish. All you need to do is to follow the rules when adding a new field. You can use the code of existing fields as an example of how to add more fields/criteria.

 

For example, the code for zip radius search is the following:

<tr>

 <td>[[FormFieldCaptions!Search Within]]:</td>                //displays the caption for the field

 <td>{search property=ZipCode}</td>                                //displays the ID of the field

</tr>

 

You can find the templates for all field types in the Field Types section.
