Where to Modify the Top Menu

You can find the template which displays the top menu in Admin Panel -> Mobile Site Layout -> Module Templates -> Menu -> menu.tpl. There you can add menu items. CSS style for the top menu is:

.menu

{

    background-color: #333333;

    float: left;

    width: 100%;

}
