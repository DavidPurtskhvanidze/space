Browse By State

 Top 

 

The 'Browse By State' block is a browsing section.

 

Administrator can display browsing section in templates using the following code construction:

{module name="classifieds" function="browse" category_id="root" fields="State" number_of_cols="3" browse_template="browse_by_state.tpl"}

{module name="classifieds" function="browse" category_id="root" fields="category_sid" number_of_cols="3" browse_template="browse_by_categories.tpl"}

 

 

Alternatively, browsing parameters can be specified in the parameters of the 'browse' function which can be set for a Site Page. When you click "Edit", you can see the parameters:

 

How to Edit the Browse By State Section

In order to edit the design of this section, you need to edit the file browse_by_state.tpl which you can find in

Admin Panel -> Site Layout -> Module Templates -> Classifieds Engine.

You can also refer to the CSS styles which are specified in the template files for browsing.

.browseByState

{

 margin: 0 20px 20px 45px;

 width: 960px;

}

 

The captions of the browsing blocks are translatable and can be modified in the templates, specified above.
