How to translate static content

 Top  Previous  Next

When visitors switch a language, you may want your Static Pages, e.g. the /about page, to be switched too and displayed translated to a specified language.

 

Please follow the below instructions:

 

1.You need to create several static pages for the "About Us" page, and you need to put there a relevant content in different languages.

E.g. you create about_en in English, about_es in Spanish and about_zh in Chinese.

 

2.Then you need to make a copy of the default.tpl file from apps\FrontEnd\templates\the_name_of_your_theme\main and paste it there and name it about.tpl.

 

3.In about.tpl find the following code:

<div class="mainContent">

 {$MAIN_CONTENT}

</div>

 

and replace {$MAIN_CONTENT} with:

 

{i18n->getCurrentLanguage assign="currentLanguage"}

{module name="static_content" function="show_static_content" pageid='about_'|cat:$currentLanguage}

 

4.Go to Admin Panel -> Site Layout -> Site Pages -> /about/ and edit it. Put the about.tpl in the "Template" field and save it.
