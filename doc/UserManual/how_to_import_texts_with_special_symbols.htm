How to import texts with special symbols

 Top  Previous  Next

Here is what you need to do:

 

1.In Excel open the file with imported listing data

 

2.Save this file as Unicode Text. In Excel menu File->Save as, in the save dialog box select "Unicode Text (*.txt)" in the "Save as type" drop-down box. Excel will save the file in UTF-16 format.

 

3.Convert file from UTF-16 to UTF-8 encoding. There are number of tools that can convert text files from one encoding to other. I myself used UltraEdit text editor to do that.

 

4.In iLister's Admin Panel go to "Import Listings", for "File type" select "CSV", for "Fields Delimiter" select "Tabulator". Set the "Not found values in DB will be" according to your preference.

 

National symbols should be imported as they appear in the text.
