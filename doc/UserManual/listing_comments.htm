Listing Comments

 Top  Previous  Next

Description

iLister provides administrator and users the ability to post comments and the replies to the comments for listings.

 

This feature allows users to communicate with each other and share opinions and experience regarding the items placed on iLister-based website.

 

The administrator has the ability to receive email notification when users post comment or comment reply if he enables the parameter "Notify on Comment/Reply Added" in Notifications Settings in Admin Panel.

 

A user can also receive such notifications in case he or she ticked the corresponding settings in the section "My Notifications" (it is available when a user is logged in):

•"Notify on New Comment Added to your Listing" — enables notification when someone posted a comment to the listing of this user;
•"Notify on New Reply Posted to your Comment" — sends an email if someone replied to the comment of this user.

 

 

How to Add a Comment

Site administrator can post a comment and reply to comments in Admin Panel on the Listing Details page and on the section "Manage Comments".

The "Manage Comments" section allows to find comments and replies of the specified listing, category, username or comment status.

 

 

Only registered users can add comments on the Listing Details page. Comments of trusted users become published immediately, unlike comments from untrusted users — they are visible for administrator, listing owner and the author of the comment, until administrator approved the comment.

 

Hidden comments and replies have the colored background. Site administrator can specify the color in Admin Panel -> Custom Settings -> hidden_comments_background_color.

 

 

Along with posting a comment, users can rate the listing.

 

 

How Users Can Add and View Replies

Each comment has the link "Reply". All replies to the comment can be visible clicking the "View all replies" link.

 

 

 

Comment Display

On the front-end Listing Details page there are only 3 last comments to the listing. All available comments can be seen on the separate "Comments" page following the link "View all comments" which is located above comments section on the Listing Details page.

 

In order to see the replies, user should click on the "View all Replies" link. The page will be expanded and all replies become visible.

 

In Admin Panel site administrator can also view all comments clicking on the "Manage Comments" link which is available for each listing at the Listing Management page when the mouse is on the listing area.

 

 

Comment Management

When the mouse is on the area of a comment, you can see comment controls aside from the comment posting date:

Reply | Hide/Publish | Edit | Delete | Make the User Untrusted/Trusted

 

Administrator can perform the following actions for each comment or reply:

•Edit — opens an edit comment page in Admin Panel
•Delete — completely deletes the comment and all it replies from the database
•Hide — the comment will be hidden for everybody excepting administrator, listing owner and the author of the comment. Administrator can hide comments which contain an improper content.
•Publish — the comment becomes visible for all visitors. Administrator should publish comments of untrusted users in order they can be seen by everybody.

 

 

How to Disable Listing Comments

Administrator can remove the ability to add comments. In order to do so, the administrator needs to "Disable" the "Listing Comments" module in Admin Panel -> System configuration -> Module Manager.
