iLister Overview

iLister is a universal platform for creating classifieds websites, which provides you with extensive, flexible and comfortable software tools for browsing by categories and listing search. iLister was optimized for search engines (SEO friendly) and facilitates indexing of iLister-based website by the search engines. iLister as well includes tools to manage users and user groups and to build and manage site's content. Using iLister you can rapidly develop a powerful, unique and fully-functional classifieds service in any area of economy or leisure.

 

This manual contains detailed description of general system concepts, Admin Panel components and instructions how to use them to help you to start with creating iLister-based website.
