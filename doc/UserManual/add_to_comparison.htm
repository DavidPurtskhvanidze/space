add_to_comparison

 Top 

add_to_comparison — adds listing to the comparison chart

 

Parameters

none

 

Templates

add_to_comparison.tpl - displays the result of adding the listing to a comparison chart

 

Local variables

•$errors - the array with the codes of errors
•$error - the key for specific error
•$smarty.request.listing_id - id of the listing
•$smarty.request.listing_type_sid - system id of the listing type
•$clearChartUrl - url to clear the comparison chart
