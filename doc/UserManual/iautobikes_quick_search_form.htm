Quick Search Forms

 Top 

 

The templates for Quick Listing Search forms are located in Admin Panel -> Site Layout -> Module Templates -> Classifieds Engine:

•quick_search_motorcycle.tpl
•quick_search_bicycles.tpl
•quick_search_accessories.tpl
•quick_search_parts.tpl

 

There you can modify the components of the Quick Search forms.

 

The code in homepage_main_content.tpl which displays Quick Search form on the homepage is the following:

<div class="quickSearchesTabs">

 <ul>

         <li><a href="#FindMotorcycleTab">[[Find a Motorcycle]]</a></li>

         <li><a href="#FindBicycleTab">[[Find a Bicycle]]</a></li>

         <li><a href="#FindAccessoriesTab">[[Find Accessories]]</a></li>

         <li><a href="#FindPartsTab">[[Find Parts]]</a></li>

 </ul>

 <div id="FindMotorcycleTab">

        {module name="classifieds" function="search_form" form_template="quick_search_motorcycle.tpl" category_id="Motorcycle"}

 </div>

 <div id="FindBicycleTab">

        {module name="classifieds" function="search_form" form_template="quick_search_bicycles.tpl" category_id="Bicycle"}

 </div>

 <div id="FindAccessoriesTab">

        {module name="classifieds" function="search_form" form_template="quick_search_accessories.tpl" category_id="Accessories"}

 </div>

 <div id="FindPartsTab">

        {module name="classifieds" function="search_form" form_template="quick_search_parts.tpl" category_id="Parts"}

 </div>

</div>

Caption

The form captions are specified in homepage_main_content.tpl in the double square brackets to give the administrator the ability to translate it.

<li><a href="#FindMotorcycleTab">[[Find a Motorcycle]]</a></li>

<li><a href="#FindBicycleTab">[[Find a Bicycle]]</a></li>

<li><a href="#FindAccessoriesTab">[[Find Accessories]]</a></li>

Criterion / Field

You can add as many fields on the search form as you wish. All you need to do is to follow the rules when adding a new field. You can use the code of existing fields as an example of how to add more fields/criteria.

 

For example, the code for list-type field is the following:

<td><span class="fieldCaption fieldCaptionState">[[$form_fields.State.caption]]:</span></td>        //displays the caption for the field

<td><span class="fieldControl fieldControlState">{search property=State}</span></td>                //displays the area with value types (i.e. tree, list, integer, string etc. which are described  here.)

 

You can add a new field, for example, Condition, as shown below:

<td><span class="fieldCaption fieldCaptionCondition">[[$form_fields.Condition.caption]]:</span></td>

<td><span class="fieldControl fieldControlCondition">{search property=Condition template=list.tpl}</span></td>

 

property=Condition specifies the ID of the field;

template=list.tpl the template which defines the way how field values are displayed on the form.

 

You can find the templates for all field types in the Field Types section.

 

Link to the Advanced Search Form

You can specify a path to a desired search form and the translatable text as a caption for the link:

<a class="searchTypeSwitch" href="{$GLOBALS.site_url}/search/motorcycles/">[[Advanced Search]]</a>
