The 'Browse Homes By State' and ''Browse Rentals By State' sections are browsing sections.

 

These wide browsing links are specified in homepage.tpl in the div mainPageSubLinks:

.mainPageSubLinks

{

    clear: both;

    padding: 10px 0 0;

}

 

When site visitor clicked a browsing section, he/she is redirected to the browsing page:

 

 

How to Edit the Homes By State Sections

In order to edit the design of these sections, you need to edit the file browse_items_and_results.tpl, which you can find in Admin Panel -> Mobile Site Layout -> Module Templates -> Classifieds Engine.

You can also refer to the CSS styles which are specified in the template files for browsing.

 

The captions of the browsing blocks are translatable and can be modified in the templates.
