Footer and Copyright

 Top 

 

The template which displays footer part and the copyright information is page_parts/footer_block.tpl, and you can find it in Admin Panel -> Site Layout -> Page Templates.

 

The CSS styles for these blocks are specified in the template and can be found in the 'base'-theme CSS file:

.copyrights

{

    height: 1.5em;

    margin: auto;

    text-align: center;

    width: 100%;

}

 

You can easily change the copyright text shown below:

<div class="copyrights">

        Powered by <a title="Classifieds software" href="http://www.worksforweb.com/classifieds-software/iLister/" rel="blank">==product_name==</a>,

        &copy; 2008&ndash;==build_year== <a href="http://www.worksforweb.com/" rel="blank">WorksForWeb</a>.

</div>

 

Please note that your copy of iLister must be eligible for un-branding in order to have the right to change the copyright info.
