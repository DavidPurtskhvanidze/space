The Android App is not a part of iLister default package for distribution. It can be purchased separately for your iLister-based website for additional cost.

 

     

 

 

 

Overview

An Android App is a special application that allows your clients to operate with your classifieds website using android smartphone. This app can be set up via admin panel. 

This application includes the iLister main features and options: 

?Search on your website has never been so easy!

 

 

•Google Map on the listing details page helps to find the location of a listing faster.

       

•Creating a new listing is now possible via our Android app without using notebooks or desktop computers

       

•Users can contact sellers via phone, sms, and email

 

How It Works For Visitors

Your website will be available for your agents and dealers all the time. The clients need to download your Android App, install it on the smartphone and that’s all. Your website is in their pockets.

The interface is very simple to understand. Clients may log in, add and modify their listings, contact sellers and search listings as well.

 

 

Android App Configuration

All the information specified in your iLister Front-End Website will appear in Android App as well. You can modify your Categories and their fields in Admin Panel, and all these changes will be applied to your Android App immediately. No need to make double configurations for the website and Android app. You need only to close the App and open it again.

You can edit the categories in Admin Panel -> Listing Management -> Categories

In order to learn more about Categories modification, please refer to the section "iLister Add a New Category" of this User Manual.

 

NOTE: The Android App is using "ZipCode" field for Google Map by default. If you're using another id(like Postcode), you need to specify the field ID in Admin Panel > System Configuration > Custom Settings > location_field_id.

 

 

Configuration of Search Forms

 

You can modify the fields for “Search Cars” in Admin Panel -> API > Search Forms -> $category -> Edit

 

If you want to add a new field that is not listed, you need to create this field in “Categories” menu firstly. After creating a new field, it will automatically appear in “Search Form Fields”.
