News, Poll and Share Your Site

 Top 

News, the Poll block and the Share Your Site section are placed on the left side under the left menu in the file at

Admin Panel -> Site Layout -> Page Templates -> page_parts/left_block.tpl.

 

Here is the code which displays these sections:

{module name="publications" function="show_publications" category_id="News" number_of_publications="2" publications_template="print_news_box_articles.tpl"}

{module name="poll" function="poll_form"}

{include file="miscellaneous:share_section.tpl"}

 

If you like to change the look of these blocks, you need the following templates:

•for News — Admin Panel -> Site Layout -> Module Templates -> Publication System-> print_news_box_articles.tpl
•for Poll — Admin Panel -> Site Layout -> Module Templates -> Poll -> poll_block.tpl
•for the Share section — Admin Panel -> Site Layout -> Module Templates -> Miscellaneous -> share_section.tpl

 

For the section with share buttons we use the AddThis tool. In order to learn how to customize the buttons and change other settings, please refer to the official AddThis help topics.

 

How to Change a Number of News

In order to change a number of news displayed at the left block, please go to Admin Panel -> Site Layout -> Page Templates -> page_parts/left_block.tpl and change the parameter 'number_of_publications'.
