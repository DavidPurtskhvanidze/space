How to Import geographic locations

 Top  Previous  Next

 Please note that only xls and csv extensions are acceptable.

 

Import xls file:

1. Create an xls file for import (see an example below):

 

2. Go to Admin Panel > System Configuration > Geographic Locations > Import geographic locations from file

Columns should be specified as numbers only. For example, column "A" should be specified as "1".

 

 

 

Import csv file:

1. Create a csv file for import (see an example below):

ID,Longitude,Latitude

00501,-73.0455,40.8151

00544,-73.0476,40.8132

 

2. Go to Admin Panel > System Configuration > Geographic Locations > Import geographic locations from file

 

 

 

Note: If you want to update locations with the same ID, you need to check "Update on Match"
