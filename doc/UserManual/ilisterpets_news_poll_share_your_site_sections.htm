Footer, Copyright and Others

 Top 

 

The template which displays the footer part, a small "About Us" section, news, Recent Tweets, poll and the copyright information is page_parts/footer_block.tpl, and you can find it in Admin Panel -> Site Layout -> Page Templates.

 

The CSS styles for these blocks are specified in the template.

footer {

    background: none repeat scroll 0% 0% #3A3A3A;

    margin: 30px 0px 0px;

    padding: 0px 0px 30px;

    color: #AAA;

}

 

.copyright-share {

    margin-top: 30px;

    padding-top: 20px;

    border-top: 1px solid #999;

}

 

You can easily change the copyright text shown below:

<div class="copyright">

           {include file="miscellaneous^powered.tpl"}

 <br>

 &copy; 2006&ndash;2014 <a href="http://www.worksforweb.com/">WorksForWeb</a>

</div>

 

The "Powered By" text can be modified in Admin Panel -> Site layout -> Module Templates -> Miscellaneous -> powered.tpl

 

 

About Us

The About Us text and the image can be also adjusted in page_parts/footer_block.tpl:

<div class="about-us">

 <h2>[[About Us]]</h2>

 <img src="{url file='main^about_us.png'}">

 <p>

         [[You can change this text to any desired wording. In order to do so please go to the Admin Panel, then click "Site Layout" then "Page Templates" and then click on the "Edit" button across the "page_parts\footer_block.tpl" item in the table.]]

 </p>

 <span class="readMore"><a href="{$GLOBALS.site_url}/about/">[[Read more]]</a></span>

</div>

 

Please put a new image to /modules/theme_boostrap_base/main/_files/about_us.png.

 

News, Poll and Share Your Site Sections

If you like to change the look of these blocks, you need the following templates:

•for News — Admin Panel -> Site Layout -> Module Templates -> Publication System-> print_news_box_articles.tpl
•for Poll — Admin Panel -> Site Layout -> Module Templates -> Poll -> poll_block.tpl
•for the Share section — Admin Panel -> Site Layout -> Module Templates -> Miscellaneous -> share_section.tpl

 

Here is the code which displays these sections:

{module name="publications" function="show_publications" passed_parameters_via_uri="" category_id="News" number_of_publications="2" publications_template="print_news_box_articles.tpl"}

oIn order to change a number of news displayed at the footer, please go to Admin Panel -> Site Layout -> Page Templates -> page_parts/footer_block.tpl and change the parameter 'number_of_publications'.

{module name="poll" function="poll_form"}

{include file="miscellaneous^share_section.tpl"}

 

 

For the section with share buttons we use the AddThis tool. In order to learn how to customize the buttons and change other settings, please refer to the official AddThis help topics.

 

Please note that your copy of iLister must be eligible for un-branding in order to have the right to change the copyright info.
