Custom Settings

 Top  Previous  Next

Custom Settings are the constants which are used in templates and which values change rarely. The administrator can create Custom Settings in Admin Panel -> Custom settings.

 

There are some default Custom Settings that are used in iLister:

•listing_currency
•transaction_currency
•number_of_last_listings_for_rss
•color_for_highlihgted_listing
•credit_name
•hidden_comments_background_color
•site_name
•uri_of_listing_details_page_on_mobile_frontend

 

All of them are used in templates as follows {$GLOBALS.custom_settings.custom_setting_name}:

<div class="search_result_item" {if $listing.feature_highlighted}style="background:{$GLOBALS.custom_settings.color_for_highlihgted_listing}"{/if}>

 

The difference of Custom Settings from System Settings is described below:

1.administrator can create his/her own Custom Setting;
2.Custom Settings are used in templates rather then System Settings, which are used both in templates and in scripts;
3.you can delete a Custom Setting and this will not break the operation of you site;
4.administrator can change the value of Custom Settings.
