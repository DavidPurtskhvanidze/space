Configuration

The configuration contains 2 steps:

1.Getting the Google Maps API key from Google
2.Specifying the API key in the Admin Panel

 

 

Getting the Google Maps API key from Google

•login to the https://code.google.com/apis/console page
•create new project
•activate Google Map API v3 via left menu -> Services -> Google Maps API v3 = On. Please pay attention that Google limits the number of free requests.
•copy the API Key from the section "API Access" located on the left:

 

 

Specifying the Google Maps API key in Admin Panel

•open the Admin Panel, the System Configuration / System Settings section
•paste the API key:

•click Save.
