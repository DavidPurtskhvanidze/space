Running Installation Script

 Top  Previous  Next

iLister has an installation script which helps you to install iLister application to your server.

 

You can run the script in a browser with a path like http://www.yoursite.com/install.php or http://yoursite.com/subfolder-where-iLister-installed/install.php

All you need to do is to fill in all fields of the installation form and then follow the instructions of the installation script, if any.

 

Installation script will tell you if any of minimal system requirements are not supported on your server so you can ask the server administrator to enable them.

 

 

On the example above you can see that all installation steps are listed on the left side.

 

The progress is also shown above the caption of the current step on the right. Please do not be afraid of the large number of the last installation step: most part of steps pass unnoticeably.

The steps which were passed successfully are marked with the green color, skipped steps are colored with orange. However, there are some steps which require mandatory action and cannot be skipped. In this case the script will inform you about this and will not go further until the action is performed.

 

At the bottom part of the script you can see the process log. It shows how the installation goes and there you can track if something goes wrong. This helps to correct server configuration, if needed.

 

Please feel free to contact the Tech Support Dept. if you have any questions during the installation.
