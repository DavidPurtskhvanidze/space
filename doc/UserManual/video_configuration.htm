Video Configuration

In order to be able to upload videos to listings, the site Administrator needs to configure Google Developers Console and Admin Panel.

 

In Google Developers Console: https://console.developers.google.com/

1. Create new project

2. Enable YouTube Data API v3 in APIs & auth > APIs

 

 

3. Set a Product name in APIs & auth > Credentials > OAuth consent screen

 

 

4. Add OAuth 2.0 client ID in APIs & auth > Credentials > Credentials

 

 

5. Set the following:

Application type - Web application

Authorized redirect URIs - http://YOU_SITE_URL/user/openid_oauth_login/

http://YOU_SITE_URL/admin/system/field_types/google_auth/

 

You'll get client ID and client secret. We'll use this information in Admin Panel.

 

 

In Admin Panel:

1. In System Configuration > System Settings > OAuth & OpenID you need to set the following:

- Client ID for Google = client ID that was generated earlier

- Client secret for Google = client secret that was generated earlier

- Google app name = Google Project name

2. Click on "Get Access Token" in Admin Panel > System Configuration > System Settings > Social Networks > Upload Video to YouTube > Upload Video Status disabled.

3. Click on "Accept" button

 

 

All videos will be uploaded to YouTube by task scheduler.

 

 

NOTE: You need to enable "Is Video Allowed" in Admin Panel > User Management > Membership Plans > Plan > Edit Package.
