Footer and Copyright

 Top 

Please note that your copy of iLister must be eligible for un-branding in order to have the right to change the copyright info.

 

 

The template which displays footer and the copyright information is page_parts/footer_block.tpl, and you can find it in Admin Panel -> Mobile Site Layout -> Page Templates.

 

The CSS styles for these blocks are specified in the template.

#PageFooter

{

    background-color: #666666;

    clear: both;

    margin-top: 20px;

}

 

.copyrights, .copyrights a, .copyrights a:visited, .copyrights a:hover {

    color: #BBBBBB;

}

 

and

.copyrights {

    font-size: 70%;

    line-height: 20px;

    text-align: center;

}

 

You can easily change the copyright text shown below in page_parts/footer_block.tpl:

<div class="copyrights">

 Powered by <a href="http://www.worksforweb.com/classifieds-software/iLister-Pets/" title="iLister Pets &mdash; Vehicle classifieds software">==product_name==</a>,

 &copy; 2006&ndash;==build_year== <a href="http://www.worksforweb.com/">WorksForWeb</a>.

</div>
