Sponsored Listings

 Top  Previous  Next

Description

The Listing Package option called "Sponsored Listings" allows listing owner to lift his/her listing among other listings on the search results page. Such sponsored ads appear above organic search results (i.e. the results which are relevant to the search criteria) on the search results page. Sponsored listings are rotated on the search results page; their change places in sponsored turn. However, when a website visitor manually sorts the search results, the sponsored status is not considered as a priority and all listings are sorted in accordance to sorting criteria.

The Sponsored feature is a great promotion tool for the listing owner, and a great money making function for the administrator.

When enabled, in addition to an upper position on the search results page, these listings are marked with the "Sponsored" label and thus stand out against other ordinary ads.

 

Administration Manual

The administrator can configure the following options of the Sponsored Listings feature via the listing package management interface in the Admin Panel:

•Number of Days for Sponsored Listing — the lifetime of the listing sponsored status (in days). The sponsored status does not expire in case it was activated by the website administrator. Empty value equals to 0 (zero).
•Price for Sponsored Listing — the price for sponsored feature (specified in credits, if the Credit System is enabled). If the price equals zero (0), a listing can get the sponsored status for free.

 

The administrator has the ability to enable the sponsored feature for any listing via the Edit Listing interface in the Admin Panel. If the administrator checks the "Sponsored" checkbox for the listing, this option is activated for the listing regardless of whether it's been paid for in the listing package.

 

The "Sponsored" label can be modified using CSS styles in /modules/listing_feature_sponsored/apps/FrontEnd/templates/_files/design.css

 

User Experience

The sponsored listing option is available for any active listing. If the option was not enabled for the Listing Package by default, the "Make Sponsored" link will be displayed in the listing description on the "My Listings" page. In case the sponsored option was enabled by default, i.e. the checkbox was ticked in the settings of a Listing Package, then no link will be available to activate it until it expires after the specified number of days.

 

 

After clicking the link, the listing owner will be offered to make a payment equal to the cost of this option set by the administrator for this listing package. If the price for this option is set to 0, this feature will be activated automatically without any further prompts to make a payment.

 

Listings that have the sponsored option enabled are labeled "Sponsored" on the search result page.
