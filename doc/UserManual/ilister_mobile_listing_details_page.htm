Listing Details Page

 Top  Previous  Next

The template which displays Listing Details information is default_view.tpl which can be found in

Admin Panel -> Mobile Site Layout -> Module Templates -> Classifieds Engine -> category_templates/display/default_view.tpl

 

 

Listing Caption

Listing caption ({$listing}) is built from the "Listing Caption" category setting and the listing fields specified in the following code:

<h1>

{if $listing.Sold.isTrue}

 <span class="fieldValue fieldValueSold">[[SOLD]]!</span>

{/if}

{$listing}

</h1>

 

You can specify any listing field to compose a caption which helps you to create a SEO-friendly title for this page.

 

Rating

Rating is displayed by the code

<span class="fieldValue fieldValueListingRating">{display property="ListingRating"}</span><br /> 

 

The template which is used for rating-type fields is rating.tpl in Admin Panel -> Mobile Site Layout -> Module Templates -> Classifieds Engine. There you can see the images that are used for rating stars, and you can easily change them.

 

Listing Image

In order to change the section with the first listing image, you need to modify the file category_templates/display/default_view.tpl in Admin Panel -> Mobile Site Layout -> Module Templates -> Classifieds Engine. The CSS style that defines the look of the image section is the following:

.listingDetails .summary .fieldValuePictures

{

    display: block;

    margin: 0 0 2px;

    text-align: center;

}

 

 

How to Add a New Field

By default, iLister displays all fields of the listing category in the foreach cycle under Description in the Listing Info block:

{foreach from=$form_fields item=form_field}

 

However, you may want to add a new field which will be displayed in a different way. You can do this as follows:

{foreach from=$form_fields item=form_field}

 {* The following code excludes some fields from being displayed *}

 {if !(

            $form_field.id == 'featured'

         || $form_field.id == 'NewField'                //Exluding your new field from the foreach cycle

...

         )}

 <span class="fieldCaption fieldCaption{$form_field.id}">[[$form_field.caption]]:</span> 

 <span class="fieldValue fieldValue{$form_field.id}">{display property=$form_field.id}</span>

 <br />

 {/if}

{/foreach}

{if $listing.NewField}

 <tr>

         <td>

                 <span class="fieldCaption fieldCaption{$form_field.id}">[[$form_field.caption]]:</span>

                 <span class="fieldValue fieldValue{$form_field.id}">[[$listing.NewField]]</span>        //Adding your new field to the Listing Details page

         </td>

 </tr>

{/if}

 

You need to change "NewField" to the ID of your new listing field.

 

 

 

Listing Details Sublinks

Listing Details sublinks direct site visitor to the pages with additional listing information such as pictures, map, seller information and others. The links are included via the file category_templates/display/subpages_links.tpl located in Admin Panel -> Mobile Site Layout -> Module Templates -> Classifieds Engine:

{include file="classifieds:category_templates/display/subpages_links.tpl" currentPageId="details" listing=$listing}

 

Listing Pictures

In order to change the page with all listing pictures, you need to modify the file category_templates/display/display_pictures.tpl in Admin Panel -> Mobile Site Layout -> Module Templates -> Classifieds Engine. The CSS styles that define the look of the image section, are specified in the display_pictures.tpl file.

.pictures .picturesContainer, .pictures .pictureContainer

{

    margin-top: 5px;

    text-align: center;

}

 

Larger Image

When visitor clicks on a thumbnail image on the Listing Pictures page, he/she is redirected to the page with the larger image for the selected picture.

The page contains picture controls which allow visitor to page pictures or to come back to the page with all listing pictures.

 

 

In order to change the look of this page, please modify the template category_templates/display/display_picture.tpl in  Admin Panel -> Mobile Site Layout -> Module Templates -> Classifieds Engine. There you can find all CSS styles regulating this section.

 

 

Google Map Details

The Google Map details are retrieved from the following data:

{getCoordsByAddress address=$listing.Address|cat:", "|cat:$listing.City|cat:", "|cat:$listing.State assign=geoData}

{if empty($geoData) && !empty($listing.ZipCode)}{assign var=geoData value=$listing.ZipCode}{/if}

{if !empty($geoData)}

 <img src="http://maps.google.com/maps/api/staticmap?size=300x300&markers={$geoData.latitude},{$geoData.longitude}&sensor=false" />

{/if}

 

Here is what is happening:

1.{getCoordsByAddress address=$listing.Address|cat:", "|cat:$listing.City|cat:", "|cat:$listing.State assign=geoData}

iLister creates the variable geoData with the data from listing fields Address, City and State. You can change this set of fields to your own.

2.{if empty($geoData) && !empty($listing.ZipCode)}{assign var=geoData value=$listing.ZipCode}{/if}

In case geoData is empty (i.e. Address, City and State were not filled for this listing) and there is a ZipCode for this listing, then the ZipCode value is passed to geoData.

3.{if !empty($geoData)}

{include file="category_templates/display/listing_details_googlemap.tpl" geoData=$geoData}

{/if}

Then, if there is any information in geoData (either from Address, City and State or from the ZipCode), iLister displays the block with the map which shows the location based on the data from geoData.

 

The map div for Google Map section in specified in the file category_templates/display/display_map.tpl in  Admin Panel -> Mobile Site Layout -> Module Templates -> Classifieds Engine.

 

Comments Section

Comments section is connected to the listing details page through display_comments.tpl that can be found in

Admin Panel -> Mobile Site Layout -> Module Templates -> Classifieds Engine -> category_templates/display/display_comments.tpl

{module name="classifieds" function="display_comments" limit=3 results_template="comments.tpl" QUERY_STRING="listing_sid[equal]="|cat:$listing.sid}

 

The CSS style for the comments section is <div class="comments">.

 

Seller Information

The seller block starts with the code <div class="sellerInfo">

You can add more information about the seller to this section from User Profile fields of the User Group. Afterwards you need to add new fields to the file category_templates/display/display_seller_info.tpl in  Admin Panel -> Mobile Site Layout -> Module Templates -> Classifieds Engine.
