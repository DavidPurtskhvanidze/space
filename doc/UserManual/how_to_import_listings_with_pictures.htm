How to import listings with pictures

 Top  Previous  Next

You can import listings with pictures as follows:

 

1.place the files with pictures to your /admin/pictures/ folder and set 777 permissions to the folder;

 

2.create an xls file for import (see an example below), where columns are field IDs;

 

3.put the names of pictures to a "pictures" column like the following:

pictures/mazda6face.jpg;pictures/mazda6back.jpg;pictures/infinity_fx45_03_06.jpg;pictures/acura_legend(1989).jpg;

(without spaces and separated by semicolon)

 

4.make an import: Admin Panel -> Listing Management -> Import Listings.
