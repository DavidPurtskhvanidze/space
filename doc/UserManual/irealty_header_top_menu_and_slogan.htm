Header, Top Menu and Slogan

 Top 

Header is the upper part of the page.

 

 

You can view header.tpl in Admin Panel -> Site Layout -> Page Templates -> page_parts/header_block.tpl.

 

 

How to Change Header

The look of the header is specified in the header block in the CSS file which you can find at in Admin Panel -> Site Layout -> Themes -> irealty_bootstrap  -> Edit design.css

header

{

    you css code here

}

 

 

Where to Modify the Top Menu

You can find the template which displays the top menu in Admin Panel -> Site Layout -> Module Templates -> Menu -> top_menu.tpl. There you can add menu items.

 

 

Slogan

You can change the slogan text in Admin Panel -> Site Layout -> Module Templates -> Miscellaneous -> logo.tpl

 

<div class="pull-left logo">

    <a href="{$GLOBALS.site_url}/">[[iRealty Classifieds]]</a>

</div>

<div class="pull-left slogan hidden-xs">[[Real Estate Demo Site]]</div>

 

You can add the logo image as well with the help of Configuration Wizard.

 

Please check which style is used for the header of your theme in Admin Panel -> Site Layout -> Page Templates -> page_parts/header_block.tpl.
