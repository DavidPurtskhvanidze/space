WYSIWYG Editor

 Top  Previous  Next

WYSIWYG editor allows users to enter the rich content into text area fields (e.g. seller comments field for listings)  to enhance presentation of their listing.

 

Administration manual

In order to enable WYSIWYG-editor go to the Admin Panel System Settings and check the "Enable WYSIWYG Editor" checkbox.

 

Warning: If  your "User Input Escaping" is set to "Escape to all HTML tags" the WYSIWYG editor generated content will be escaped and will show as HTML code, but not as rich-formatted content. In order to use WYSIWYG editor, please select "Remove potentially unsafe tags with HTML purifier" option.
