Hosting Requirements

Please find below the list of main system requirements for a hosting:

•Apache Web server HTTD 1.3 or higher
•PHP 5.4
•PHP allow_url_fopen option enabled
•ionCube Loader
•mod_rewrite
•PHP safe_mode set to "Off"
•MySQL 5.0.3 or higher
•Recommended min 256Mb dedicated PHP RAM mem limit, 512Mb for Feed Import Addon
•.htaccess support
•Imagick, СURL, BCMath, zlib and iсonv libraries
•WGЕT utility
•PHP register_globals set to "Off"
•PHP mail option enabled
•MySQLi
•PHP Data Objects (PDO) set to "On"

 

Please note that there can be more settings and/or components of the hosting/server environment which are necessary for the iLister software to work properly, but only main configuration is described.
