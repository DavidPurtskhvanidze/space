Overview

SmsCoin is a micro-payment system that provides easily accessible payments through SMS billing for online services.

 

Currently an SMS aggregator SmsCoin provides premium SMS payment solutions across 98 countries supporting hundreds of mobile carriers worldwide. There is a choice of several short codes operating with different SMS cost in each country. In vast majority of the countries connection of mobile payments does not require any additional approval, short codes become available for your project immediately following an SMS payment service set up.

 

SmsCoin is called a micro-payment system due to the fact that it allows making payments of a limited value amount for each transaction. This limitation relates to different SMS payment policies of mobile providers. Please learn more about tariff rates at SmsCoin website. You can also contact SmsCoin support service to find out how to correctly configure chargers on your website to stay within allowed transaction value.

 

You can learn more about SmsCoin on their official website at http://smscoin.com/en/.

 

 

Configuring sms:bank Interface

Although SmsCoin provides several services, we've integrated the sms:bank service with iLister.

 

In order to use sms:bank on your website you need to have a valid SmsCoin account. You can learn how to register an account on the SmsCoin Registration Page.

 

When the account is registered, you need to create a bank on the Purses Page.

 

After login and proceeding to the "Purses" page please click the link "Add". You will be redirected to the "Add sms:bank" page.

 

Please fill in the following fields:

•Name — a desired name for the bank.
•Secret code — the secret key which you should specify for this bank.
•Countries — please select the countries available for your payments.
•Success URL — the URL on your website where SmsCoin redirects users in case of successful payment. Please insert there http://www.yourwebsite.com/system/payment/callback/smscoin/
•Success URL method — select "Post"
•Fail URL — the URL on your website where SmsCoin redirects users in case of payment cancellation. Please insert there http://www.yourwebsite.com/system/payment/callback/smscoin/
•Fail URL method — select "Post"
•Result URL — SmsCoin payment processor URL on your website. Please insert there http://www.yourwebsite.com/system/payment/callback/smscoin/result (please note that there is no / at the end)
•Result URL method — select "Post"
•Timeout — 0
•Active — check the option to make the bank active.

 

After the form is completed, please click the button "Save".

 

 

 

SmsCoin Payment Gateway Configuration

The SmsCoin Payment Gateway can be configured in the Admin Panel -> Payment Management -> Payment Gateways -> SmsCoin -> Configure

 

 

 

 

Please fill in the configuration form with the following data:

•SmsCoin Bank ID — the number of your bank, which you can see in your bank settings at the SmsCoin "Manage sms:bank" page.
•SmsCoin Bank Secret — the secret code which you specified in your bank settings.

 

Please keep in mind that there is no testing mode in SmsCoin. For testing transaction please contact SmsCoin technical support service.
