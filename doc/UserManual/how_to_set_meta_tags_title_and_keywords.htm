How to set TITLE and KEYWORDS meta tags

 Top  Previous  Next

In templates you have a code like the following in listing_details.tpl or default_view.tpl:

{title}[[$listing.PropertyName:raw]], [[$listing.City:raw]], [[$listing.type.caption:raw]], [[$listing.PropertyType:raw]]{/title}

{keywords}[[$listing.PropertyName:raw]], [[$listing.City:raw]], [[$listing.District:raw]], [[$listing.type.caption:raw]], [[$listing.PropertyType:raw]]{/keywords}

{description}[[$listing.PropertyName:raw]], [[$listing.type.caption:raw]], [[$listing.City:raw]], [[$listing.PropertyType:raw]], [[$listing.nearestMRT:raw]] {/description}

 

 

Or if you want your /browse_by_state/ Site Page to display the "State" field value in these tags, you need to do the following:

1.Add the following code in browse_homes_by_state.tpl:

{title}[[$listing.State:raw]]{/title}

{keywords}[[$listing.State:raw]]{/keywords}

{description}[[$listing.State:raw]]{/description}

 

2.Here at http://yoursite.com/admin/site_pages/?action=edit_page&uri=/browse_homes_by_state/

in the Keywords and Description fields add:

Browse Homes by State

 

3.If you like to display title and keywords in html for search robots, you need to put the following code into the <head> container of homepage_main_content.tpl, default.tpl and blank.tpl:

<meta name="keywords" content="{$KEYWORDS}" />

<meta name="description" content="{$DESCRIPTION}" />

<title>{$TITLE}</title>

 

 

The page source will display:

<head>

<meta name="keywords" content="Browse Homes by State Louisiana" />

<meta name="description" content="Browse Homes by State Louisiana" />

<title>Browse Homes by State Louisiana</title>

</head>
