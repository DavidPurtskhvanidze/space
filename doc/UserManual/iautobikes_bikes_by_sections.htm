Bikes By... Sections

 Top 

 

The 'Motorcycles By Make' and 'Motorcycles By Body Style' sections are browsing sections.

 

Administrator can display browsing section in templates using the following code construction:

{module name="classifieds" function="browse" category_id="Motorcycle" fields="State" number_of_cols="4" browse_template="browse_motorcycles_by_state.tpl"}

 

In the theme iauto_bikes_green browsing sections are shown in tabs:

<div class="browsingTabs">

 <ul>

         <li><a href="#BrowseMotorcycleTab">[[Motorcycle by State]]</a></li>

         <li><a href="#BrowseBicycleTab">[[Bicycle by State]]</a></li>

 </ul>

 <div id="BrowseMotorcycleTab">

     {module name="classifieds" function="browse" category_id="Motorcycle" fields="State" number_of_cols="4" browse_template="browse_motorcycles_by_state.tpl"}

 </div>

 <div id="BrowseBicycleTab">

        {module name="classifieds" function="browse" category_id="Bicycle" fields="State" number_of_cols="4" browse_template="browse_bicycles_by_state.tpl"}

 </div>

</div>

 

How to Edit the Browsing Sections

In order to edit the design of these section, you need to edit the files browse_motorcycles_by_state.tpl and browse_bicycles_by_state.tpl, which you can find in

Admin Panel -> Site Layout -> Module Templates -> Classifieds Engine.

You can also refer to the CSS styles which are specified in the template files for browsing.

 

The captions of the browsing blocks are translatable and can be modified in the template homepage_main_content.tpl:

<li><a href="#BrowseMotorcycleTab">[[Motorcycle by State]]</a></li>

<li><a href="#BrowseBicycleTab">[[Bicycle by State]]</a></li>
