The Facebook Comments is an add-on and is not a part of iLister default package for distribution. It can be installed separately for your iLister-based website. Please request the installation conditions from our Sales Dept.

 

Overview

iLister has integrated Facebook Comments Plugin that lets people comment on content of your site(listings, user profiles) using their Facebook account.

 

Users can also post their comments on Facebook.

 

 

How To Insert Facebook Comments plugin

 

You can insert the Facebook comments on Listing details page using the following code:

{module name="facebook_comments" function="display_comments" url="{page_url id="listing"}|cat:$listing.id"}

 

For user profile you can use the following code:

{module name="facebook_comments" function="display_comments" url="{page_url id='users'}"|cat:$user.id} 

 

Please note that you need to set Facebook ID and secret in Admin Panel -> System Settings. You can find more information in Facebook, Twitter, Google and OpenID Login section.
