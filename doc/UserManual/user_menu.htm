user_menu

 Top 

user_menu - displays user menu

 

Parameters

user_group_sid - the SID of a user group

user_menu_template - the user menu template for the specified user group. For example, you can use private_seller_menu.tpl for the "Private Sellers" user group.

 

Templates

user_menu.tpl or the template specified in properties of a user group - the template for the logged-in user default menu

 

Local variables

$GLOBALS.site_url -  the global variable containing a full URL of the site
