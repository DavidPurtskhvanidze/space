Availability Calendar

 Top  Previous  Next

Availability calendar allows to display and manage reservation periods (e.g. availability for rent) or display availability of the vehicle for certain activities (e.g. maintenance).  Availability calendar is represented in the application as a listing field type. Thus, administrator can add availability calendars to any listing type (or all listings types) via the Admin Panel.

 

The availability calendar is displayed on the listing details page. Reserved periods on the calendar are highlighted on the calendar.

 

Managing an availability calendar

Availability calendar includes unlimited number of reservation periods. Each period has the following properties: start date, end date, and a comment. Listing owner and website administrator can edit availability calendar: add periods to and delete periods from the calendar.

To edit availability calendar go to edit listing page and click the "Edit Calendar" link below the calendar.

 

 

A calendar edit page opens:

To add a reservation to the calendar fill in the "Book Property" form and click the "Add" bottom. Text entered to the "Comment" field will not be shown on the listing details page. It can be used by the listing owner for any relevant information pertaining to the reserved period. If period period data was entered correctly a confirmation messages appears:

In case of error a corresponding error message appears above the "Add New Period" form.

 

Below the form is a table of periods reserved so far.  Listing owner (or administrator) can delete the period by clicking the "Delete" link in the corresponding row. Alternatively, the owner can check the checkboxes next to periods and click on the "Delete" button.

 

 

Visitor experience

When a visitor opens a listing details page it displays the availability calendar for the listing:

 

 

Visitor is able to see the reserved period highlighted on the calendar. By default calendar is set to display current month and next two month. A visitor can browse calendar forth and back by clicking the arrow controls on the right and left of the upper bar of the calendar.

 

The "Book Property" link opens a booking request form:

 

 

Using this form a visitor can send a booking request to the listing owner. The booking request will be sent to the listing owner's email.
