The CAPTCHA (Completely Automated Public Turing test to tell Humans and Computers Apart, see wikipedia article) is used to ensure a human is using the form and protects the system from automated message dispatch and registrations (SPAM users and messages).

 

For these puprposes the system uses a KCAPTCHA library. The captcha configuration, such as symbols used to draw CAPTCHA, CAPTCHA string length etc. can be changed in the /vendor/kcaptcha/kcaptcha_config.php file.

 

Currently, captcha is implemented on Contact Us, Contact Seller, Tell Friend, Contact User, Report Inappropriate Content and Registration pages.

For all pages where captcha is implemented, you can easily turn it on or off. For this, in the Admin Panel go to System Configuration -> System Settings section.

On this page, in the Captcha Settings, check or uncheck the check boxes to turn on or off captcha display on corresponding pages.
