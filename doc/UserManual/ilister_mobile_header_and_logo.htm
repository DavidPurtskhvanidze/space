Header and Logo

 Top  Previous  Next

Header is the upper part of the page. There you can see logo and language switcher.

 

 

The look of the header is specified in the header div in the CSS file which you can find at /modules/main/apps/FrontEnd/files/{theme_name}/design.css:

.header

{

    background-color: #1370D9;

}

 

Other components (logo and language switcher) are applied on top of the header background and they are specified in Admin Panel -> Mobile Site Layout -> Page Templates -> default.tpl.

The CSS style for language switcher is the following:

.header .languageSelector

{

    color: #FFFFFF;

    float: right;

    margin: 10px 10px 0 0;

}

 

How to Change Header

You can change the color of the header in the .header style which is shown above.

Header size also can be changed, the default 100% width is specified and inherited from the parent theme "mobile_base" :

.header

{

    float: left;

    width: 100%;

}

 

We do not recommend to change the width, unless your site uses a completely new custom theme which is not inherited from iLister product themes.

 

Please check which style is used for the header of your theme in Admin Panel -> Mobile Site Layout -> Page Templates -> default.tpl.

 

Logo

The logo of iLister-based website is located in the header of each page.

 

The file name for a logotype is logo.png and it is located on your server at the iLister installation folder

/modules/main/apps/FrontEnd/templates/_files/logo.png.

Please note that different themes can use different logo files and thus you need to make sure that your {theme_name} folder correlates with the theme which logo should be changed.

 

You can learn more about themes and templates in the Template System and Theme Management section.

 

You can design your own logo file and upload it to the location specified above, under the same name, i.e. logo.png.

 

In order to upload your own logo.png, you need to connect to your server via FTP or via Control Panel (please ask your hosting administrator if your Control Panel has the option to work with file transferring).

 

The position of logo file is specified by CSS style, like the following example:

.header .logo

{

    float: left;

}

 

You can add more parameters there like:

.header .logo

{

 display:block;

 left:5px;

 position:absolute;

 top:5px;

}

 

You can find the CSS file for your theme at /modules/main/apps/FrontEnd/files/{theme_name}/design.css
