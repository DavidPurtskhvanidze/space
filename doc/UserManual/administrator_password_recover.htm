Administrator Password Recovery

 Top  Previous  Next

Description

The administrator password recovery feature allows website administrator to restore the forgotten password from the login form in the Admin Panel.

 

In order to restore the password, the website administrator should specify his username on the form and the website will send him an email with a link to the page where he can change his password.

 

If the specified admin username exists, the website sends an email with the password recovery link to the email listed in the "Notification Email" field in Admin Panel -> System Settings.

 

Otherwise you will see a warning reporting the incorrect admin username.

 

To complete the password change you should follow the link provided in the email.
