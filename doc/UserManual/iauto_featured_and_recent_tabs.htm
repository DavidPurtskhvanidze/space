Featured Listings And Recent Ads Tabs

 Top 

In order to edit the look of the Featured Listings section you need to edit the template featured_listings.tpl in Admin Panel -> Site Layout -> Module Templates -> Featured Listing (\modules\theme_iauto_tabs_blue\listing_feature_featured\featured_listings.tpl).

 

The way how featured ads are displayed can be specified by the code in homepage_main_content.tpl:

{module name="listing_feature_featured" function="featured_listings" featured_listings_template="featured_listings.tpl" number_of_rows="2" number_of_cols="2"}

 

For the Recent Ads the code is similar:

{module name="recent_listings" function="display_recent_listings" number_of_rows="2" number_of_cols="2"}

 

You need to place this code to the template of the page where you would like to see the Featured Ads section.

 

You can change the number of rows and number of columns in the code to display as many ads as you wish.

{module name="listing_feature_featured" function="featured_listings" featured_listings_template="featured_listings.tpl" number_of_rows="1" number_of_cols="4"}

will display

Please note that it's available for non-responsive themes only.

In responsive themes the number of rows and number of columns are calculated automatically depending on screen resolution.

 

Also you can specify the category for Featured Listings and Recent Listings blocks by setting "category_id":

{module name="listing_feature_featured" function="featured_listings" featured_listings_template="featured_listings.tpl" number_of_rows="2" number_of_cols="2" category_id="Car"}

{module name="recent_listings" function="display_recent_listings" number_of_rows="2" number_of_cols="2" category_id="Car"}

}

}

and their child styles.
