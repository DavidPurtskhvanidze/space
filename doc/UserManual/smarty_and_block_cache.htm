Smarty Cache

Smarty caches the information, which is often displayed, in order to reduce the time required for displaying the site page.

It has the link "Clear Cache" which deletes temporary cache files of Smarty template processor.

You may need to use this feature when you suspect that the changes which you made in templates are not actually displayed on your website.

When you modified a template, Smarty should update related caching file, but sometimes it does not. In this case you see an old file, and not an actual new version of the file. That is why you can clear cache and this will resolve the conflict between the real file and its old cached version.

 

Block Cache

The Block Cache option is designed to speed up loading of the front page.

The contents of various blocks on the front page is cached (e.g. Recent Ads, Browse by block, etc.).

When the cache option is ON, data in those blocks is cached and the homepage load speed increases.

You can set cache refresh intervals independently for each block in the relevant templates.

If you want to add cache to block, you need to add cached=true cacheLifeTime='1D' where

1M - 1 month

1D - 1 day

1H - 1 hour

1 - 1 minute

The result will look like

{module cached=true cacheLifeTime='60' name="recent_listings" function="display_recent_listings" number_of_rows="1" 

number_of_cols="4" do_not_modify_meta_data=true}

 

There is also an ability to turn on/off the block cache of homepage by clicking on "Block cache on the main page".

 

The Administrator can clear cache and turn on/off the homepage block cache in Admin Panel > System Configuration > System Settings > Cache
