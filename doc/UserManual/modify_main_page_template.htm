Modify Main Page Template

 Top  Previous  Next

Main page is your home page - it is opened first when your site is opened.

 

In accordance with default iLister Site Pages settings, main page uses default.tpl as a template, which surrounds the content of the home page.

 

The home page of iLister has the URI / and the title "home".

 

In Admin Panel -> Site Pages -> / you can see that the "Template" parameter is empty which means that it uses default.tpl as a template:

 

Below you can see how the page is built on the front-end of the site. Default.tpl includes:

•headerMenuBlock with top menu, user greeting and language selector
•sloganBlock div
•mainContentBlock div including:
ohomepage_main_content.tpl - the template file which displays the content for the home page. You can see it is used for the / Site Page:
▪module is 'main'
▪function is 'display_template'
▪template_file is 'homepage_main_content.tpl'
othe "quickSearchAndNewsWrapper" div with the Quick Search form and News section
oAdsAndFacebookWrapper including:
▪Featured and Recent listings block,
▪the Facebook Like Box,
▪Google AdSense section and
▪browsing tabs
•footerBlock with footer_block.tpl, copyright, News box and the sections Poll and Share.

 

 

You can learn more on how iLister builds pages in the "Page Building" section.

The "Template Identification" feature will help you to find out which template is used to display the parts of any iLister page.
