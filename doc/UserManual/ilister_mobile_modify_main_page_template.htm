Modify Main Page Template

 Top  Previous  Next

Main page is your home page - it is opened first when your mobile site is opened.

 

In accordance with default iLister mobile Site Pages settings, main page uses default.tpl as a template, which surrounds the content of the home page.

 

The home page of iLister has the URI / and the title "home".

 

In Admin Panel -> Mobile Site Layout -> Site Pages -> / you can see that the "Template" parameter is empty which means that it uses default.tpl as a template:

 

Below you can see how the page is built on the front-end of the site. Default.tpl includes:

•header div  with logo and language switcher
•menu div including menu.tpl with top menu
•content div including:
ohomepage.tpl - the template file which displays the content for the home page. You can see it is used for the / Site Page:
▪module is 'main'
▪function is 'display_template'
▪template_file is 'homepage.tpl'
oquick_search.tpl representing Quick Search form
oFeatured Ads section
oBrowse Categories browsing block in the mainPageSubLinks div
•footer_block.tpl with copyright

 

 

You can learn more on how iLister builds pages in the "Page Building" section.

The "Template Identification" feature will help you to find out which template is used to display the parts of any iLister page.
