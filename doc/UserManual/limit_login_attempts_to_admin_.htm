Limit login attempts to Admin Panel

 Top  Previous  Next

Limiting the failed login attempts will lock a user out if they entered the wrong password more than the specified time. They will be locked out for a specified time. You can control the settings from your admin panel.

 

 

Secure admin logging settings

 

You can configure admin logging settings in Admin Panel -> System Configuration -> System Settings -> General -> Secure admin logging

 

Number of failed login attempts - Number of failed authorizations after which the system will block further logging into the admin panel.

Failed login monitoring interval (mins) - Time interval during which the system counts failed login attempts

Login suspension duration (mins) - Time interval during which any further login attempts to the admin panel are impossible
