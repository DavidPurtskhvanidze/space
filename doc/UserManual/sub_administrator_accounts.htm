The Sub-Administrator Addon module is not a part of iLister default package for distribution. It can be installed separately for your iLister-based website. Please request the installation conditions from our Sales Dept.

 

Overview

Sub-Administrator Addon allow the administrator to create sub-administrator users with limited access to the Admin Panel functions. This feature is very handy if you plan to have several specialists working on different parts and areas of your website. Thus your marketing / promotion specialist will not be able to access the listing management area, and a listing moderator cannot pip to the payments section.

 

 

Create New Sub-Administrator Account

You can add a new sub-administrator in the Admin Panel -> System Configuration -> Sub-Administrators. On the right side you can adjust the functions which are accessible for this administrator.

 

Administrator can see the list of sub-administrators and manage their settings.

 

When a sub-administrator logged in the Admin Panel, he can see the allowed section on the left menu only. If a sub-administrator tries to open a restricted page, the website will display him a warning and will not let him go to that page.
