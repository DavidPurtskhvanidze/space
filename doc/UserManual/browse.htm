browse

 Top 

browse —  browsing listings by any field of string, list, integer, geo or tree types.

 

Parameters

category_id — optional parameter, an ID of a category which listings will be considered for browsing

fields — a list of listing fields and a sub-category level which are available for browsing

number_of_levels — maximum number of browsing levels visitors will see

browse_template — template for browsing (by default, browse_items_and_results.tpl )

default_sorting_field — the field that will be used for default sorting for browsing results

default_sorting_order — the order that will be used for default sorting order (ASC or DESC: ascending or descending) for browsing results

default_listings_per_page — the default number of listings per page for browsing results

 

 

Templates

browse_items_and_results.tpl - displays the page for browsing by the "BodyStyle" field

 

Local variables

•$GLOBALS.site_url - the global variable containing a full URL of the site
•$current_page_uri - the URI for the "Browse by Body Style" site page (e.g. /browse/, /browse_by_make_model/)
•$TITLE - the title for the page in a browser
•$browse_navigation_elements - the array of the navigation elements for browsing
•$element.metadata - contains the domain for internationalization, e.g. for the MakeModel field the domain is "Property_MakeModel"
•$element.caption - the caption (name) of an element
•$element.uri - the URI for the page for the particular element
•$browseItems - the elements for browsing, selected by criteria (field for browsing) and browse level
•$browseItem.url - SEO friendly values to pass them to browsing parameters
•$browseItem.caption - the caption of the element
•$browseItem.count - the number of listings matching browsing criteria
•$columnCount - the number of columns
