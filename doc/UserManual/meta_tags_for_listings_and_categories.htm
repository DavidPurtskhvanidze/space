Meta Tags for Listings and Categories

 Top  Previous  Next

iLister became even more SEO friendly: the meta tags Description, Keywords and Title can be added for a listing and a category, which makes them unique, and thus, more attractive for web crawlers and search engines.

 

Category Meta Tags

The website administrator can add specific meta tags to a category on the Edit Category page in the Admin Panel.

Category meta tags are used on the Category Browsing and Search Results pages.

 

 

Listing Meta Tags

The website administrator can add target meta tags to a listing on the Edit Listing page in the Admin Panel.

 

Listing meta tags are used on the Listing Details page. If there are no specific meta tags for a listing, the website uses the tags from the listing category.

 

If a Site Page like Listing Details or Search Results has its own meta tags, all tags are used: the tags of a Site Page plus Category tags, plus Listing Tags.
