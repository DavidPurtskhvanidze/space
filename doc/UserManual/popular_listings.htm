The Popular listings is an add-on and is not a part of iLister default package for distribution. It can be installed separately for your iLister-based website. Please request the installation conditions from our Sales Dept.

 

Overview

Popular listings addon allows website administrator to display the most popular listings depend on number of views.

 

 

 

 

Popular Listings Settings

Website administrator can set the following settings for Popular Listings addon:

•popular_listings_template - the template that will display the popular listings block (by default it's popular_listings.tpl)
•number_of_rows -  the number of rows that you want to display (works only in non-responsive themes)
•number_of_cols - the number of columns that you want to display (works only in non-responsive themes)
•category_id - if specified, the listings will be displayed from specified category (by default it's root category)

 

 

For responsive themes you can specify number of displayed listings in the block by setting number_of_rows and number_of_cols:

number_of_rows * number_of_cols = number of displayed listings

 

 

In order to insert the Popular Listings block to a page (template), please use the code as follows:

{module name="popular_listings" function="display_popular_listings" number_of_rows="2" number_of_cols="4"}
