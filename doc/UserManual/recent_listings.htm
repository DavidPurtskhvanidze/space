Recent Listings

The section with recent listings is made to demonstrates the activity of your iLister-based website. It is located on the main page, and you can easily change its look, location and other parameters. You can learn more about this section in the User Manual article at iLister Site Customization -> Recent Ads Section.

 

 

 

The Recent Listings Module

The look of the recent listing section can be easily changed in the design.css file related to a theme like /application/modules/{theme_name}/main/_files/design.css

The styles can be the following:

.homepageMainContent .recentListings

.homepageMainContent .recentListings > h1

.homepageMainContent .recentListings > ul

.homepageMainContent .recentListings .item

.homepageMainContent .recentListings .item .fieldValue.fieldValuePictures

.homepageMainContent .recentListings .item .details .fieldValue.fieldValuePrice,

.homepageMainContent .recentListings .item .details .fieldValue.fieldValueRent

.homepageMainContent .recentListings .item .details .fieldValue.fieldValuePrice > a

 

You can find the module at Admin Panel -> System Configuration -> Module Manager among other modules. There you can disable the Recent Listings module if you do not want to use it.

 

In order to insert the recent listing section to a page (template), please use the following code:

{module name="recent_listings" function="display_recent_listings" number_of_rows="1" number_of_cols="4"}

 

Also you can specify the category for Recent Listings block:

{module name="recent_listings" function="display_recent_listings" number_of_rows="1" number_of_cols="4" category_id="Car"}
