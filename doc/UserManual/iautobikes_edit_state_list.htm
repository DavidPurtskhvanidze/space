Edit List of States

 Top 

Conceptually, iLister has two types of lists with States:

1.States for listings — you can see it among Common Fields in Admin Panel -> Categories -> follow the "Edit Common Fields" link
2.States for users — you can find it in the Dealers user profile fields at Admin Panel -> User Groups -> Dealer -> click the "Edit User Profile Fields" link.

 

 

Both lists can be managed in the same way (as others list-type fields).

 

How to Add a State

In order to add a new state, please click the link "Edit List Values". You can add values to a list-type field in two ways:

1.you can manually fill in the form "Add a new Value" which interface is intuitive and you can easily fill in the form.

 

When you added a new value, you can change its position using the arrows UP or DOWN.

 

2.alternatively, you can import the file with several states following the link "Import data from file".

 

The file can be either in CSV or XLS format.
