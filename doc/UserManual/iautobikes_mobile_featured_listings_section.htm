Featured Listings Section

 Top 

 

In order to edit the look of the Featured Listings section you need to edit the template featured_listings.tpl in Admin Panel -> Mobile Site Layout -> Module Templates -> Classifieds Engine.

 

The way how featured ads are displayed can be specified by the code in homepage.tpl:

{module name="listing_feature_featured" function="featured_listings" featured_listings_template="featured_listings.tpl" number_of_rows="1" number_of_cols="3"}

 

You need to place this code to the template of the page where you would like to see the Featured Ads section.

 

You can change the number of rows and number of columns in the code to display as many ads as you wish.

{module name="listing_feature_featured" function="featured_listings" featured_listings_template="featured_listings.tpl" number_of_rows="3" number_of_cols="1"}
