User Management

 Top  Previous  Next

The "Manage Users" page of the Admin Panel provides access to the following functionality:

•view all registered users of the site,
•select users basing on specified criteria,
•open and edit the user profile page,
•perform following bulk actions with users:
oactivate and deactivate,
osend activation letter,
odelete users,
ochange user's User Group,
omake users trusted or untrusted.

 

 

 

 

Using the page's interface

On the top of the page there is a filter form that allows to filter user by certain criteria:

•Username — filters by username or a part of the username;
•Email — filters by users' email or any part of the email;
•User Group — selects users of the specified user group;
•Membership Plan — allows to include into search results the users subscribed to a certain Membership Plan. Besides all available Membership Plans there is an option which will pick out the users who have not subscribed yet, or with the expired subscription. It is a great pro-active tool to communicate with new users to encourage them to start using your website.
•Registration Date — allows to select users registered within certain date range. Dates are accepted in the date format specified for the default website's language.

 

You can leave all of the filter fields blank if you would like to see all the registered users. Click on the "Search" button to select users based on the specified criteria.

 

The search results are shown in the form of the table.

 

Per-user actions

The user search results interface contains controls that perform different actions, that can be applied to the users found. Each search result line has the following action controls section. This section is revealed when the mouse is on the area of the specified user:

 

The last column of the table contains links to actions that can be applied to the user in the corresponding row. The actions include:

•"Activate" or "Deactivate" — if user never clicked on the activation link sent to the user's email, the user remains inactive after registration. Inactive users are not able to log in into the website. In case the email is lost or user is unable to active his/her account with activation link, administrator can activate such user manually in by clicking "Activate" link. This link is displayed only for inactive users. For active users a "Deactivate" link is display, clicking on which will make the user inactive.
•Send Activation Letter — allows to send welcome email with activation link to user's email. This link help in case user did not get the email send to him/her automatically after registration.
•Edit — opens an Edit user page. This page allows to edit user profile details, change user's password, expire user's membership plan.
•Delete — deletes the user from the site completely. All listings belonged to the user are deleted from the database along with the user removal. Please use with caution.

 

Bulk actions

There are check boxes in the very first column for the search results table. Checking this checkbox allows selection of users for bulk action. For convince, the "check all / uncheck all" links are provided on the top and on the bottom of the table.

Each bulk action that is available is represented with a corresponding button on top of the page and on the bottom of the page. There are following actions available:

•Activate and Deactivate — allowing to activate or deactivate all selected users. Please see description of the per-user "Activate" or "Deactivate" actions for details.
•Send Activation Letter — sends welcome email with activation link to all selected users.
•Change User Group — allows to change the current User Group for the selected user. When administrator clicks the link a pop-up window is opened allowing to choose a new User Group among available User Groups listed there:
oIn case new User Group has a Membership Plan which user has been already subscribed to (within the old User Group), the user subscription stays intact with this Membership Plan. Otherwise the existing user subscription is deleted and the user is prompted to subscribe to a new Membership Plan next time he post a listing.
oCommon user fields stay intact, while old User Group specific fields are cleared. The user is asked to fill in new User Group specific fields next time he is at the "My Profile" section.
oThe user can also change the User Group using the link "Change User Group" in the "My Profile" section. When he chooses new User Group which does not have his old Membership Plan available, the site warns the user that his old subscription will be deleted.
•Delete — deletes all selected users. All users listings are deleted from the database along with users removal.
•Make Trusted and Make Untrusted — sets or removes the "trusted" flag on all selected users, more information on users' "trusted" status can be found in the Trusted Users section of the manual.

 

Edit User Profile Page

The Edit User page in Admin Panel opens when administrator clicks on the "Edit" button on the Manage Users page.

 

 

On the Edit User page the following functionality is offered:

•Edit user profile fields
•Change user password
•Change user User Group
•Expire user subscription (plan)

When edit user page is opened the fields "Password" and password confirmation are left blank. This however, does not mean that user does not have password or user's password is blank. If you save users profile leaving the password fields blank, users' password won't change. To change the password, enter new value into this field and save profile.

 

 

Expiring user subscription

If administrator clicks on "Expire user's plan" button user's contract is deleted from the database. This is different from normal contract expiration. In this case the listing of subscription-based user will not be deactivated. Neither auto-extension will be attempted nor notification will be sent to the user.

 

Deleting the user contract is useful when user would like to change subscription before the actual expiration of current subscription. It is especially helpful if the current user's subscription is eternal.

 

After user contact has been deleted from the database, user can go the "Subscription" page and choose membership plan again.

 

Change the User Group

The site administrator can change current User Group for a user. Clicking the "Change User Group" link at the User Info page will open a small modal window where the administrator can select new User Group for the user.
