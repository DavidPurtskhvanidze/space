Cars By... Sections

 Top 

 

The "Cars By Make", "Cars By State" and "Cars By Body Style" sections are browsing sections.

 

Administrator can display browsing section in templates using the following code construction:

{module name="classifieds" function="browse" category_id="Car" fields="MakeModel" number_of_cols="3" browse_template='browse_by_make_model.tpl'}

{module name="classifieds" function="browse" category_id="Car" fields="State" number_of_cols="3" browse_template='browse_by_state.tpl'}

{module name="classifieds" function="browse" category_id="Car" fields="BodyStyle" number_of_cols="2" browse_template='main_browse_by_body_style.tpl'}

 

This code is specified in \modules\theme_iauto_tabs_blue\main\homepage_main_content.tpl which you can access in the Admin Panel -> Site Layout -> Page Templates -> homepage_main_content.tpl

 

How to Edit the Cars By... Sections

In order to edit the design of these section, you need to edit the files browse_by_make_model.tpl, browse_by_state.tpl and main_browse_by_body_style.tpl, which you can find in the Admin Panel -> Site Layout -> Module Templates -> Classifieds Engine.

You can also refer to the CSS styles which are specified in the template files for browsing.

 

The captions of the browsing blocks are translatable and can be modified in the templates.

 

A new icon for Body Style must be placed to the folder /modules/theme_iauto_base/main/_files/ and the file must be named of the exact name of a body style and has the extension .png. For example, the icon for the body style Van is named Van.png.
