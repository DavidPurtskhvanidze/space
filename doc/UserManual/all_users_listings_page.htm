All User's Listings Page

 Top  Previous  Next

This page displays all listings of the user similar to search results. On top of the page user profile is displayed. This page is created with the help of the user_details function of the users module and a special template named “user_details_with_all_ads.tpl”   set via the “display_template” parameter. This template includes a block of listing search results which contains all active listings of the user.

 

The URL of this page is SEO-friendly and is configurable. It may include user’s last name or company name, and any other fields available in the user profile.  The profile fields that appear in the URLs are configured via the relevant templates.

 

Site Visitor Experience

When users click on the “View All Listings” on the user search results page, a new page opens that contains all listings of the user underneath the user profile info. The all listings block contains the same brief info as displayed on the listing search results pages.

User search results are paginated, with controls to flip pages and set a total number of results per page located above and underneath the results block.
