How to translate the field "Description"

 Top  Previous  Next

If you want to have the field Description (for iLister) or SellerComments (for iAuto and iRealty) in several languages and display a translation in accordance to a selected language, you can do the following:

 

1.Create the "CommentsRU" and "CommentsES" field of the "text" type

 

2.Add these fields to all related input forms:

<tr>

        <td>{input property=Comments}</td>

</tr>

<tr>

        <td>{input property=CommentsRU}</td>

</tr>

<tr>

        <td>{input property=CommentsES}</td>

</tr>

 

For iLister, these fields will be automatically added to apps\FrontEnd\templates\base\classifieds\category_templates\input\default_input.tpl

 

3.Specify the condition to display a proper field on the Listing Details page:
•For iLister
1)put the following code to the default_view.tpl file in the very beginning:

{i18n->getCurrentLanguage assign="currentLanguage"}

 

2)Then replace the line:

<span class="fieldValue fieldValueDescription">{$listing.Description}</span>

 

with the following code:

{if $currentLanguage eq 'en'}
<span class="fieldValue fieldValueDescription">{$listing.Description}</span>
{elseif $currentLanguage eq 'ru'}
<span class="fieldValue fieldValueDescription">{$listing.DescriptionRU}</span>

{elseif $currentLanguage eq 'es'}
<span class="fieldValue fieldValueDescription">{$listing.DescriptionES}</span>
{/if}

 

3)Also add the line:

|| $form_field.id == 'DescriptionRU'

 

after:

|| $form_field.id == 'Description'

 

4)Save the file.

 

•For iAuto or iRealty
1)put the following code to the default_view.tpl file in the very beginning:

{i18n->getCurrentLanguage assign="currentLanguage"}

2)Then replace the code:

<span class="fieldCaption fieldCaptionSellerComments">[[FormFieldCaptions!Seller Comments]]:</span>

<span class="fieldValue fieldValueSellerComments">{$listing.SellerComments}</span>

 

with the below lines:

{if $currentLanguage eq 'en'}
<span class="fieldCaption fieldCaptionSellerComments">[[FormFieldCaptions!Seller Comments]]:</span>
<span class="fieldValue fieldValueSellerComments">{$listing.Comments}</span>
{elseif $currentLanguage eq 'ru'}
<span class="fieldCaption fieldCaptionSellerComments">[[FormFieldCaptions!Seller CommentsRU]]:</span>
<span class="fieldValue fieldValueSellerComments">{$listing.SellerCommentsRU}</span>

{else $currentLanguage eq 'es'}
<span class="fieldCaption fieldCaptionSellerComments">[[FormFieldCaptions!Seller CommentsES]]:</span>
<span class="fieldValue fieldValueSellerComments">{$listing.CommentsES}</span>
{/if}

 

3)Save the file.
