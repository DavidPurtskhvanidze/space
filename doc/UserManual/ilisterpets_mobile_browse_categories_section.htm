How to Edit the Browse Categories Section

In order to edit the design of this section, you need to edit the file browse_items_and_results.tpl, which you can find in Admin Panel -> Mobile Site Layout -> Module Templates -> Classifieds Engine.

You can also refer to the CSS styles which are specified in the template files for browsing.

 

The captions of the browsing blocks are translatable and can be modified in the templates.
