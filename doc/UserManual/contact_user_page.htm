Contact User Page

 Top  Previous  Next

The contact user page contains a form visitors can fill out and submit. The contents of the form is emailed to the user.

 

In order to set up a page with the contact user form, please follow the standard user page setup procedure and use the "contact_form" function of the "users" module.  To enable CAPTCHA on this form, check the “Captcha in the Contact User Form” checkbox in the System Settings of the Admin Panel.

 

Site Visitor Experience

 

When the visitor opens a contact user page, it displays the Contact User form. If the visitor followed the "Contact Seller" link at the Listing Details page, he would see the modal window with the contact form instead of being redirected to a separate page.
