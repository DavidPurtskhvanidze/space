Listing Caption

 Top  Previous  Next

Description

iLister provides administrator the ability to specify the caption of a listing for each category. Specified listing caption is displayed on the Listing Details page. The setting is called "Listing Caption" and can be found on the configuration page of a category.

 

This setting is inherited by child categories. This means that site administrator can define one Listing Caption for a parent category which will be inherited by all sub-categories; and re-declare the Caption for child categories which demand different Listing Caption.

 

How to use Listing Caption

The administrator can use the variable {$listing} in the template which displays the Listing Details page (please refer to the section "Template Identification").

{title}{$listing|cat:""|strip_tags:false}, [[$listing.type.caption:raw]]{/title}

{keywords}{$listing|cat:""|strip_tags:false}, [[$listing.type.caption:raw]]{/keywords}

{description}{$listing|cat:""|strip_tags:false}, [[$listing.type.caption:raw]]{/description}

<h1>{$listing}</h1>

 

The {$listing} variable will be displayed on the Listing Details page in accordance to the configuration of "Listing Caption" setting in the properties of listing category.
