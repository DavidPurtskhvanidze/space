Quick Search Form

 Top  Previous  Next

The Quick Search form allows visitors to find listings with desired criteria with one click from your homepage.

The template for Quick Search form is located in Admin Panel -> Site Layout -> Module Templates -> Classifieds Engine -> quick_search.tpl.

There you can modify the components of the Quick Search form.

 

Criterion / Field

You can add as many fields on the search form as you wish. All you need to do is to follow the rules when adding a new field. You can use the code of existing fields as an example of how to add more fields/criteria.

 

For example, the code for keywords-type field is the following:

<li class="fieldControl textBox fieldKeywords inputValuePlaceholder" rel="[[FormFieldCaptions!Keywords]]">

 {search property="keywords" template="string_with_autocomplete.tpl"}        //displays the area with value types (i.e. tree, list, integer, string etc. which are described  here.)

</li>

 

You can find the templates for all field types in the Field Types section.

 

 

Link to the Advanced Search Form

You can specify a path to a desired search form and the translatable text as a caption for the link:

<a href="{$GLOBALS.site_url}/search/">[[Advanced search]]</a>

 

 

Other Components On The Right

Other components on the right of the Quick Search form are specified in /modules/theme_ilister_tabs_blue/main/page_parts/single_column_main_content_block.tpl.

 

These items use the divs news (/modules/theme_ilister_tabs_blue/main/_files/design.css) and newsBlock (/modules/main/apps/FrontEnd/templates/_files/design.css) :

.quickSearchAndNewsWrapper .news

{

    color: #FFFFFF;

    float: right;

    width: 420px;

}

.newsBlock

{

    margin: 10px 0 0;

}
